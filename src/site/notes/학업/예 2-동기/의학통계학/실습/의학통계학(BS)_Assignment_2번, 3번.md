# 문항 2번

## 1. 단변량 회귀분석 결과 (문항 2-2)

> [!info] **분석 목적**
> 각 독립변수가 종속변수($XMAN\ Power$)와 가지는 개별적인 상관관계를 파악함. 

| 분석 변수 | 회귀식 | 해석 및 유의성 |
| :--- | :--- | :--- |
| **고혈압 여부** | $XMAN = 99.271 + 7.848 \times HTNG$  | 고혈압($HTNG=1$)인 경우 Power가 평균 **7.848 증가**. 유의한 양의 상관관계 ($p < 0.001$).  |
| **체중** | $XMAN = 81.649 + 0.296 \times Weight$  | 체중 1kg 증가 시 Power **0.296 증가**. 유의한 양의 상관관계 ($p < 0.001$).  |
| **신장** | $XMAN = 97.168 + 0.020 \times Height$  | 신장 1cm 증가 시 Power 0.020 증가하나, **통계적으로 유의하지 않음** ($p = 0.474$).  |
| **혈중 크레아티닌** | $XMAN = 90.425 + 12.784 \times Cr$  | Cr 1 단위 상승 시 Power **12.784 증가**. 유의한 양의 상관관계 ($p < 0.001$).  |
| **ALT** | $XMAN = 95.804 + 0.211 \times ALT$  | ALT 1 상승 시 Power **0.211 증가**. 유의한 양의 상관관계 ($p < 0.001$).  |
| **AST** | $XMAN = 94.052 + 0.273 \times AST$  | AST 1 상승 시 Power **0.273 증가**. 유의한 양의 상관관계 ($p < 0.001$).  |

## 2. 모델별 수정 결정 계수 요약 (문항 2-3)

> [!check] **수정 결정 계수 (Adjusted $R^2$) 비교**

- **체중**(0.031)과 **ALT**(0.030) 모델이 상대적으로 높은 설명력을 보이며 유의함. 
- **신장**(0.000) 모델은 설명력이 없으며 통계적으로 유의하지 않음.

## 3. 영화 선호도(Filmlike) 가변수 회귀분석 (문항 2-4)

> [!note] **분석 설정**
> - **기준 집단**: 사극(4) 선호군 
> - **입력 변수**: FILM1(액션), FILM2(로맨스), FILM3(판타지), FILM5(호러) 

### **가변수와 종속변수 간 관계 해석**
1. **액션(FILM1)**: 사극 선호군 대비 Power가 평균 **8.219 높음** ($p < 0.001$, 유의함). 
2. **로맨스(FILM2)**: 사극 선호군 대비 Power가 평균 **4.475 높음** ($p < 0.001$, 유의함). 
3. **판타지(FILM3)**: 사극 선호군과 통계적으로 유의한 차이 없음 ($p = 0.068$). 
4. **호러(FILM5)**: 사극 선호군과 통계적으로 유의한 차이 없음 ($p = 0.653$).
