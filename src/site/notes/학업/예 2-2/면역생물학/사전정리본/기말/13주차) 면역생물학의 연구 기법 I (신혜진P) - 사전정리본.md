> [!abstract] 강의 요약: 면역학 연구기법 I (Flowchart)
>
> 이번 강의는 크게 **"시각화(Seeing) -> 분석(Analyzing) -> 조작(Manipulating)"**의 흐름으로 진행되었습니다.
>
> 1.  **Visualization (단백질과 세포 보기)**
>     * **고정된 조직:** 항체와 효소(HRP)를 이용한 **IHC**로 위치와 문맥 파악.
>     * **세밀한 구조:** 금 입자를 이용한 **IEM**으로 해상도 극대화.
>     * **살아있는 세포:** 형광(Fluorescence)과 **공초점/다광자 현미경**을 이용해 3D 및 실시간(Intravital) 관찰.
>
> 2.  **Genomic Analysis (유전자와 염색질 분석)**
>     * **위치 확인:** **FISH**를 통해 염색체 전좌(예: CML)나 유전자 거리를 시각화.
>     * **상호작용:** **ChIP**으로 단백질-DNA 결합을, **3C**로 DNA-DNA 상호작용을 분석.
>
> 3.  **Genetic Manipulation (유전자 조작 및 모델링)**
>     * **유전자 가위:** **CRISPR-Cas9**을 이용해 특정 유전자를 자르거나(KO), 조절(CRISPRi).
>     * **동물 모델:** **Transgenic** 및 **Knockout** 마우스 생성.
>     * **정밀 제어:** **Cre-Lox 시스템**을 이용해 "원하는 조직에서, 원하는 때에" 유전자를 조작.

> [!example] 핵심 키워드 정리 (Key Terms & Concepts)
>
> ### 1. Imaging & Microscopy
> * **IHC vs ICC:** 조직(Context) vs 세포(Localization).
> * **HRP & DAB:** 갈색(Brown) 발색의 표준. (cf. AP & pNPP는 Yellow).
> * **Stokes Shift:** 흡수 파장보다 방출 파장이 더 길다(에너지 손실).
> * **Fluorophores:** FITC(Green/525nm), PE(Red-Orange/Bright!), APC(Red/660nm).
> * **Confocal vs Multiphoton:**
>     * **Confocal:** **Pinhole**로 초점면 외 빛 차단 -> 선명한 상.
>     * **Multiphoton:** **장파장(적외선)** + 2개 광자 동시 타격 -> **Deep penetration** & **Low photobleaching**.
>
> ### 2. Chromatin & Genomics
> * **FISH:** 3D 구조 및 유전자 거리 측정 (CML의 **BCR-ABL** 전좌 확인).
> * **ChIP (Chromatin IP):** 단백질-DNA 결합 분석. (Enrichment = Input 대비 비율).
> * **Histone Markers:**
>     * Active: **H3K4me3**, Acetylation.
>     * Inactive: **H3K9me3**, **H3K27me3**.
>
> ### 3. CRISPR-Cas9
> * **Components:** Cas9(가위) + **sgRNA**(crRNA+tracrRNA) + **PAM**(5'-NGG-3').
> * **Repair:** **NHEJ**(Indel/KO) vs **HDR**(Correction).
> * **dCas9 (Dead Cas9):** **D10A & H840A** 변이. 자르지 않고 붙기만 함. (CRISPRi, Epigenetic editing 등에 응용).
>
> ### 4. Animal Models
> * **3Rs:** Replace, Reduce, Refine.
> * **Strains:**
>     * **Syngeneic (동계):** 유전자 동일 (Inbred).
>     * **Congenic:** 특정 유전자좌(Locus) 하나만 다름.
> * **Gene Targeting (Knockout):**
>     * **Homologous Recombination** 이용.
>     * **Selection:** **Positive(NeoR)** + **Negative(TK/Ganciclovir)**.
> * **Cre-Lox System (족보):**
>     * **Cre:** 자르는 효소 (Tissue-specific promoter).
>     * **loxP:** 잘리는 위치 (Target gene flanking).
>     * **Inducible:** **Cre-ER + Tamoxifen** (시간 조절 가능).

> [!NOTE] Pages 1-9: 항체 매개 시각화 기초 (IHC/ICC) 및 발색 원리
> 면역학 연구기법의 첫 단추는 **"보이지 않는 단백질을 항체로 염색해서 눈으로 보는 것"**입니다. 가장 기초적이면서도 임상/연구 모두에서 필수적인 파트입니다.
>
> > [!INFO] PAGE 1-2: 강의 목표 및 개요
> > * **표지 이미지(P.1):** 공초점 현미경 등으로 촬영된 세포들의 화려한 이미지가 보입니다. 앞으로 배울 기술들이죠.
> > * **핵심 목표(Goals, P.2):** 단순히 실험 방법만 외우는 게 아닙니다.
> >     1.  면역학 관련 문헌을 읽고 **방법론의 원리**를 이해해야 합니다.
> >     2.  연구자가 선택한 실험 기법이 **"적절한지"** 판단할 수 있는 비판적 사고가 필요합니다.
> >     3.  나만의 면역학적 의문을 해결할 **실험 설계(Experimental Design)** 능력을 길러야 합니다.
>
> > [!INFO] PAGE 3-4: 항체 매개 시각화 (개요)
> > * **주제:** 항체 매개 미세구조 시각화 (Antibody-Mediated Microscopic Visualization).
> > * **두 가지 큰 줄기 (P.4):**
> >     1.  **IHC/ICC:** 효소가 달린 항체(Enzyme-Conjugated Antibodies)를 이용해 **고정된 조직(Fixed Tissues)**의 이미지를 만듭니다.
> >     2.  **Immunoelectron Microscopy (면역전자현미경):** 금 입자(Gold Beads)를 사용하여 항체가 결합한 항원을 전자현미경으로 봅니다.
>
> > [!IMPORTANT] PAGE 5: ICC vs IHC (정의 및 차이)
> > 이 두 용어의 정의를 정확히 구분하세요. 대상(Target)이 다릅니다.
> > * **Immunocytochemistry (ICC, 면역세포화학법):**
> >     * 대상: **Cells** (배양 세포, 현탁액 세포).
> >     * 목적: 세포 내 특정 단백질(항원)의 **위치(Localization)**를 시각화.
> > * **Immunohistochemistry (IHC, 면역조직화학법):**
> >     * 대상: **Tissue section** (조직 절편).
> >     * 목적: 생체 조직(Biological tissues) 내에서 세포들의 분포와 항원의 위치를 **공간적 문맥(Spatial Context)** 속에서 파악.
> >     * *피라미드 그림을 보세요. Cells -> Tissues -> Organs 순으로 올라갑니다. Tissues 레벨을 보는 게 IHC입니다.*
>
> > [!IMPORTANT] PAGE 6: 효소와 기질 (Enzymes & Substrates) **[상세 암기]**
> > 항체 끝에 달린 **효소(Enzyme)**가 **기질(Substrate)**을 만나 **색깔(Color)**을 내는 화학 반응입니다. 표의 내용을 달달 외우세요.
> >
> > **1. HRP (Horseradish Peroxidase) 효소 기질:**
> > * **DAB (3,3'-diaminobenzidine):** **Brown (갈색)**. 가장 대표적입니다.
> > * **TMB (3,3',5,5'-tetramethylbenzidine):** **Blue** (흡광도 650nm). 주로 ELISA에 씁니다.
> > * **ABTS:** **Blue-green** (흡광도 410nm).
> > * **Luminol:** 발광(Light) 반응.
> >
> > **2. AP (Alkaline Phosphatase) 효소 기질:**
> > * **pNPP (p-Nitrophenyl Phosphate):** **Yellow** (흡광도 405nm).
> >
> > **3. Background (위양성 주의):**
> > * **적혈구(RBC):** Peroxidase가 많아서 HRP 사용 시 배경이 지저분해집니다.
> > * **폐포 세포(Alveolar cells):** AP가 많습니다.
> > * **유색 조직(Pigmented tissue):** 흑색종(Melanoma)처럼 원래 까만 조직은 갈색인 DAB를 쓰면 안 보입니다. 이때는 Purple, Red, Blue 계열 기질을 써야 합니다.
>
> > [!INFO] PAGE 7-8: IHC 데이터 해석 및 워크플로우
> > * **데이터 해석(P.7):**
> >     * **그림:** 제1형 당뇨병 모델(NOD mice). 파란색은 Insulin(베타세포), 갈색(Brown-red)은 CD8 T세포입니다.
> >     * **의미:** CD8 T세포가 인슐린을 만드는 베타세포를 공격(Killing)하여 파괴하는 자가면역 기전을 보여줍니다. (PD-L1과 PD-1 상호작용 관련 내용 포함)
> > * **실험 과정(P.8):**
> >     * Fixation(심장 관류) -> Dehydration -> Embedding(Paraffin/OCT) -> Sectioning -> Staining -> Mounting -> Imaging. 이 순서, 머릿속에 그리세요.
>
> > [!TIP] PAGE 9: 임상 판독 기준 (Scoring System) **[기출 포인트]**
> > 병리과에서 암 진단 시 사용하는 **HER2 IHC 판독 기준**입니다. 수치 싸움입니다.
> > * **Score 0 (Negative):** 염색 없음 혹은 **10% 미만**의 종양세포에서 불완전한 막 염색.
> > * **Score 1+ (Negative):** **10% 이상**에서 희미하게 보이나, **불완전한(Incomplete)** 막 염색.
> > * **Score 2+ (Weakly Positive):** **10% 이상**에서 약~중간 강도의 **완전한(Complete)** 막 염색.
> > * **Score 3+ (Positive):** **10% 이상**에서 **강한(Strong)** 완전한 막 염색.
> > * *핵심은 '10% 이상인가?' 그리고 '막이 완전히(Complete) 둘러싸였는가?'입니다.*

> [!NOTE] Pages 10-16: 면역전자현미경(IEM) & 형광 시각화
> 해상도(Resolution)의 끝판왕인 전자현미경과, 특정 분자를 빛나게 하는 형광 기법입니다.
>
> > [!INFO] PAGE 10: Immunoelectron Microscopy (IEM)
> > * **원리:** 항체에 색소 대신 **금 입자(Colloidal Gold Particle)**를 붙입니다. 금은 전자밀도(Electron-dense)가 높아 전자현미경에서 **까만 점(Black dots)**으로 보입니다.
> > * **다중 표지(Double Labeling):** 금 입자의 **크기(Size)**를 다르게 하여 두 가지 항원을 동시에 봅니다.
> >     * 예시 데이터: **MHC class II는 30nm**, **Class I은 15nm** 금 입자로 표지하여 세포 내 위치 관계를 분석했습니다.
>
> > [!INFO] PAGE 11: 면역형광(IF) 개요
> > * 단순히 보는 것을 넘어, 세포 내 분자(DNA 등)의 시각화, 생체 내(Intravital) 관찰까지 확장됩니다.
> > * 그림의 "Black Light Run"처럼 어둠 속에서 특정 부분만 빛나게 하는 원리입니다.
>
> > [!IMPORTANT] PAGE 12-13: 형광 원리 및 Fluorophores 스펙트럼 **[필수 암기]**
> > **1. Jablonski Diagram (P.12):**
> > * 빛을 흡수(Absorbance) -> 들뜬 상태(Excited) -> 에너지 일부 손실 -> 빛 방출(Emission).
> > * **Stokes Shift:** 방출되는 빛(Emission)은 흡수된 빛보다 항상 **파장이 깁니다(Longer wavelength)**. 에너지가 낮아졌으니까요.
> >
> > **2. 주요 형광물질 표 (P.13):** 짝을 지어 외우세요.
> > * **DAPI/Alexa 405:** Abs ~400nm / Em ~420nm (Blue). 핵 염색에 씀.
> > * **FITC (Fluorescein):** Abs **490nm** / Em **525nm** (Green). 가장 기본.
> > * **PE (Phycoerythrin):** Abs 490, 565 / Em **578nm** (Red-Orange). **밝기(Relative Brightness)가 5**로 매우 밝습니다.
> > * **APC (Allophycocyanin):** Abs **650nm** / Em **661nm** (Red).
> > * **Cy5:** Abs 650 / Em 670.
> > * *실험 설계 시 레이저 파장과 형광물질의 Absorbance가 맞는지 반드시 확인해야 합니다.*
>
> > [!INFO] PAGE 14: 형광현미경 필터 구조 (Filter Sets)
> > 원하는 파장만 골라내는 **3단계 필터링**이 핵심입니다.
> > 1.  **First barrier filter:** 광원에서 온 빛 중 특정 파장(예: Blue 450-490nm)만 통과시켜 시료를 때립니다(Excitation).
> > 2.  **Beam-splitting mirror:** 510nm 이하는 반사(Reflects), 그 이상은 통과(Transmits). 파장에 따라 길을 나눕니다.
> > 3.  **Second barrier filter:** 시료에서 나온 형광(예: Green 520-560nm)만 통과시키고, 나머지 잡광은 차단(Cuts out)하여 눈(Eyepiece)으로 보냅니다.
>
> > [!TIP] PAGE 15-16: GFP & RFP 그리고 Transgenic Animals
> > * **역사:** 1962년 Shimomura가 해파리(*Aequorea victoria*)에서 **GFP(Green)**를, 1999년 Lukyanov가 산호(Coral)에서 **RFP(Red)**를 발견했습니다. (2008년 노벨 화학상)
> > * **응용(Whole Animal):** 16페이지 사진을 보세요. **Actin promoter** 뒤에 GFP 유전자를 붙였습니다. Actin은 모든 세포에 있으니, 갓 태어난 쥐(Neonatal mice) 전체가 초록색으로 빛납니다.

> [!NOTE] Pages 17-21: 고급 이미징 (Confocal, Multiphoton, Intravital)
> 이제 **3차원(3D)**과 **살아있는(Live)** 상태를 보는 첨단 기법입니다.
>
> > [!IMPORTANT] PAGE 17: 공초점 현미경 (Confocal Microscopy)
> > * **핵심 부품:** **Pinhole (핀홀)**.
> > * **기전:** 초점면(Focal plane)에서 출발한 빛만 통과시키고, 초점면 위아래(Out of focus)에서 산란된 빛은 핀홀 주변에서 **차단**됩니다.
> > * **결과:** 흐릿함이 없는 **선명한 상**을 얻고, Z축 스캔을 통해 **3차원 영상** 제작이 가능합니다.
>
> > [!IMPORTANT] PAGE 18: 다광자 현미경 (Multiphoton Microscopy)
> > 공초점의 업그레이드 버전입니다.
> > * **원리:** **장파장/저에너지(Long wavelength, Low energy)**의 적외선 레이저를 씁니다.
> > * **Two-photon Excitation:** 광자 1개로는 에너지가 부족해서, 광자 2개가 동시에 때려야만 형광이 나옵니다. 즉, 초점이 맺힌 점(Focus spot) 외에는 아예 형광이 안 생깁니다. (핀홀 불필요)
> > * **장점:**
> >     1.  **광퇴색(Photobleaching) 최소화:** 에너지가 낮아 검체 손상이 적습니다.
> >     2.  **깊은 투과력:** 장파장은 조직 깊숙이 들어갈 수 있습니다.
>
> > [!TIP] PAGE 19-21: 생체 내 영상 (Intravital Imaging) **[데이터 해석]**
> > 살아있는 동물에서 면역세포의 움직임을 실시간으로 봅니다.
> >
> > **1. 혈관 투과성 실험 (P.19):**
> > * **설계:** 귀 진피(Ear dermis)에 **VEGFA164**(혈관내피성장인자) 주입. 혈관 내에는 **2000 kDa FITC-Dextran**(녹색) 주입.
> > * **결과:** VEGFA 주입 후 혈관 직경이 커지고, 녹색 형광이 혈관 밖으로 새어 나가는 **Vascular leakage**가 관찰됨.
> >
> > **2. 호중구의 박테리아 제거 (P.20):**
> > * **상황:** UPEC(대장균) 감염 모델.
> > * **관찰:** 0분부터 **150분**까지 촬영. 호중구(Neutrophil)가 몰려와 박테리아(형광)를 포식하여 제거(Clearance)하는 과정을 시각화.
> >
> > **3. 림프절 T세포 이동 (P.21):**
> > * **표지:** T세포는 **CFSE (Green)**로, 고내피세동맥(HEV)은 **Red**로 염색.
> > * **결과:** T세포가 HEV를 통해 림프절로 들어와 이동하는 3차원 궤적(Trafficking)을 확인.


> [!NOTE] Pages 22-27: 유전체 시각화 (FISH) 및 염색질 구조
> 유전자의 위치를 눈으로 확인하고, 유전자가 감겨 있는 히스톤의 상태를 분석하는 파트입니다.
>
> > [!INFO] PAGE 22-23: 3-D FISH (Fluorescence in situ hybridization)
> > * **정의:** 형광 RNA 탐침(Probe)을 DNA에 붙여서 염색체의 3차원적 구조와 거리를 측정합니다.
> > * **거리 측정 데이터 (P.23):**
> >     * **T cells:** $V_H$ (Red)와 $V_{H}J558$ (Green) 유전자 거리가 멉니다(Ready discriminated). 아직 재조합 전이라 그렇습니다.
> >     * **Pro-B cells:** 면역글로불린 유전자좌(Ig locus)가 **수축(Contraction)**합니다. 두 유전자(Red/Green)가 가까워져서 색이 겹쳐 보일 정도입니다. 이게 유전자 재조합의 전조 현상입니다.
>
> > [!IMPORTANT] PAGE 24: 만성 골수성 백혈병 (CML)과 필라델피아 염색체 **[병리 연계]**
> > 이 그림은 족보 중의 족보입니다.
> > * **정상:** 9번(Green)과 22번(Red) 염색체는 떨어져 있어야 합니다.
> > * **CML (Chronic Myeloid Leukemia):**
> >     * 9번과 22번 사이에 **상호 전좌(Reciprocal Translocation)**가 일어납니다.
> >     * **결과:** 9번의 **ABL** 유전자와 22번의 **BCR** 유전자가 합쳐져 **BCR-ABL** 융합 유전자가 생깁니다.
> >     * **병리:** 이 융합 단백질은 **Tyrosine Kinase 활성**이 비정상적으로 높아져 암(Tumor)을 유발합니다. (이걸 표적으로 한 약이 글리벡입니다.)
> >     * *그림 B/C를 보면 Red/Green 점이 붙어서 노란색/중간색으로 보이는 게 전좌의 증거입니다.*
>
> > [!INFO] PAGE 26-27: 염색질 구조 (Chromatin Structure)
> > * **구조:** DNA는 히스톤(Histone)을 감아 **뉴클레오솜(Nucleosome)**을 형성합니다. (Beads on a string)
> > * **히스톤 꼬리 변형 (Histone Tail Modification):**
> >     * **Acetylation (아세틸화):** 히스톤과 DNA 결합을 느슨하게 함 -> **Open Chromatin** -> **Gene Active**. (전사 기구가 접근 가능)
> >     * **Methylation (메틸화):** 주로 **Closed Chromatin** -> **Gene Inactive**. (단, 위치에 따라 다름)

> [!NOTE] Pages 28-32: 단백질-DNA 상호작용 분석 (ChIP & 3C)
> "이 단백질이 DNA 어디에 붙었니?"(ChIP)와 "이 DNA가 저 DNA랑 만났니?"(3C)를 묻는 기술입니다.
>
> > [!IMPORTANT] PAGE 28-30: ChIP (Chromatin Immunoprecipitation) **[실험 과정 암기]**
> > 단백질-DNA 결합을 분석하는 표준 기법입니다.
> >
> > **1. Workflow (P.29):**
> >     * **Cross-linking:** 포름알데히드로 단백질과 DNA를 딱 붙임(Fixation).
> >     * **Fragmentation:** DNA를 잘게 조각냄.
> >     * **Immunoprecipitation (IP):** 특정 단백질에 대한 **항체(Antibody)**로 낚아챔.
> >     * **DNA Extraction & Analysis:** 단백질을 녹여 없애고 DNA만 남겨서 PCR이나 Sequencing으로 분석.
> >     * *결과값: Enrichment = X/R (Input 대비 얼마나 농축됐는가)*
> >
> > **2. 주요 항체 리스트 (P.30 표):** *이거 구분 못하면 해석 못 합니다.*
> >     * **Active Markers (전사 활성):** H3K4me3 (Trimethyl), H3K9ac, H3K14ac (Acetylation은 주로 활성).
> >     * **Repressive Markers (전사 억제):** **H3K9me3**, **H3K27me3**. (꽉 뭉쳐서 유전자 꺼짐)
>
> > [!INFO] PAGE 31-32: 3C (Chromosome Conformation Capture)
> > 멀리 떨어진 DNA끼리의 상호작용(Looping 등)을 봅니다.
> > * **핵심 과정:** Cross-link -> **Restriction Enzyme (제한효소) 절단** -> **Ligation (연결)**.
> > * *포인트: 원래 멀리 있던 DNA라도, 단백질에 의해 묶여서 가까이 있다면 Ligation 되어 하나의 가닥으로 연결됩니다.*
> > * 결과물: Ligation products를 PCR/Sequencing하여 "아, 1번 염색체의 Promoter가 멀리 있는 Enhancer랑 붙어먹었구나"를 알아냅니다.

> [!NOTE] Pages 33-41: 유전자 가위 (CRISPR-Cas9)
> 2020년 노벨 화학상의 주인공입니다. 단순한 가위질을 넘어 유전자 조절까지 확장된 최신 경향을 파악하세요.
>
> > [!INFO] PAGE 33-35: 역사 (Timeline)
> > * **1987:** 대장균(E. coli)에서 CRISPR 서열(Clustered repeats) 최초 발견 (이땐 뭔지 몰랐음).
> > * **2012:** **Doudna & Charpentier** (노벨상 주역)가 Cas9의 DNA 절단 기전 규명.
> > * **2013:** 인간 세포(Mammalian cells)에서 게놈 교정 성공.
> > * **2020:** 노벨 화학상 수상.
>
> > [!IMPORTANT] PAGE 36-37: 작동 기전 (Mechanism) **[핵심 원리]**
> > 박테리아의 적응 면역 시스템에서 유래했습니다.
> >
> > **1. 구성 요소:**
> >     * **Cas9 Nuclease:** 가위(Enzyme).
> >     * **sgRNA (single guide RNA):** **crRNA** (Target 인식 20nt) + **tracrRNA** (Cas9 결합 구조)를 하나로 합친 것.
> >     * **PAM (Protospacer Adjacent Motif):** **5'-NGG-3'**. Cas9이 DNA를 자르려면 타겟 서열 바로 뒤에 무조건 이 NGG가 있어야 합니다. (없으면 못 자름!)
> >
> > **2. 수선 기전 (Repair Pathways):** 잘린 DNA(DSB)는 두 가지로 수선됩니다.
> >     * **NHEJ (Non-homologous end joining):** 대충 갖다 붙임 -> 돌연변이(Indel) 발생 -> **Gene Knockout (유전자 파괴)**.
> >     * **HDR (Homology-directed repair):** 정상 DNA 조각(Donor DNA)을 넣어줌 -> 그걸 보고 베껴서 수선 -> **Gene Correction (유전자 교정/삽입)**.
>
> > [!IMPORTANT] PAGE 38-39: Cas9 구조와 변형 (dCas9) **[상세 구조]**
> > **1. Cas9의 도메인:**
> >     * **RuvC Domain:** 비표적 가닥(Non-target strand) 절단.
> >     * **HNH Domain:** 표적 가닥(Target strand) 절단.
> >     * 이 두 도메인이 모두 작동해야 이중 나선이 잘립니다(DSB).
> >
> > **2. dCas9 (dead Cas9): 가위 기능을 죽인 버전**
> >     * **변이(Mutation):** **D10A** (RuvC 불활성) + **H840A** (HNH 불활성).
> >     * **기능:** DNA를 자르지는 못하고, 가이드 RNA를 따라 **붙기만 합니다(Binding only)**.
> >     * **Nickase:** 둘 중 하나만 죽이면(예: D10A만), 한 가닥만 자르는 Nickase가 됩니다.
>
> > [!TIP] PAGE 40: dCas9의 응용 (CRISPRi & Epigenetics)
> > 자르지 못하는 dCas9에 다른 효소(Effector)를 달아서 씁니다.
> > * **CRISPRi (Interference):** dCas9이 프로모터에 딱 붙어서 전사 기구가 못 오게 막음 -> **Transcription OFF**.
> > * **Epigenetic Editing:**
> >     * **dCas9 + EZH2:** EZH2는 **H3K27 Methyltransferase**입니다. 타겟 유전자의 히스톤을 메틸화시켜 영구적으로 꺼버립니다(Silencing).
> >     * **dCas9 + GFP:** 특정 유전자 위치를 형광으로 추적(Live imaging).

> [!NOTE] Pages 42-51: 동물 실험 기초 및 형질전환 동물
> "왜 쥐를 쓰는가?"부터 "어떻게 유전자를 넣는가?"까지, 동물 실험의 A to Z입니다.
>
> > [!INFO] PAGE 42-43: 동물 실험 윤리 (3Rs)
> > 연구자의 기본 소양입니다. 무조건 외우세요.
> > * **Reduce:** 사용되는 동물의 **수(Number)**를 줄입니다.
> > * **Refine:** 동물에게 가해지는 **고통/스트레스(Stress)**를 최소화하도록 실험법을 개선합니다.
> > * **Replace:** 동물 실험을 세포 실험이나 시뮬레이션 등 **다른 방법(Other methods)**으로 대체합니다.
>
> > [!IMPORTANT] PAGE 44-45: 근교계 (Inbred Strains) **[필수 암기]**
> > 실험 편차를 줄이기 위해 유전자가 99% 이상 동일한 "순계" 마우스를 씁니다.
> > * **생성법:** **형매교배(Brother-sister mating)**를 **20세대** 이상 반복합니다.
> > * **주요 Strain 특성 (P.45 표):** *이거 족보입니다.*
> >     * **BALB/c:** 방사선 감수성(Sensitivity), 하이브리도마(Hybridoma) 제조에 사용.
> >     * **C57BL/6:** 방사선 조사 후 간암(Hepatoma) 발생 빈도 높음, 높은 보체(Complement) 활성.
> >     * **C3H/HeJ:** 유선 종양(Mammary tumor) 발생 빈도 높음.
> >     * **AKR/J:** 백혈병(Leukemia) 발생 빈도 높음.
> >     * **CBA/N:** X-linked 면역결핍(Immunodeficiency).
>
> > [!INFO] PAGE 46-47: Congenic Strains & Adoptive Transfer
> > * **Congenic (동계):** 딱 **하나의 유전자좌(Locus)**만 다르고 나머지 모든 유전자가 동일한 쥐입니다. 특정 유전자의 기능만 콕 집어 연구할 때 씁니다.
> > * **Adoptive Transfer (양자 전달):** 같은 Inbred strain끼리는 거부 반응이 없죠? 그래서 한 쥐의 림프구(Lymphocyte)를 분리해서 다른 쥐에게 주사하여 면역 반응을 전달하는 실험입니다.
>
> > [!IMPORTANT] PAGE 48-51: Transgenic Animals (형질전환) 생성법
> > 외부 유전자(Transgene)를 인위적으로 삽입하는 기술입니다.
> > * **핵심 과정 (Microinjection):**
> >     1.  수정란(Fertilized egg)의 **전핵(Pronucleus)**에 DNA를 직접 찔러 넣습니다.
> >     2.  이 수정란을 **가임신(Pseudopregnant)** 대리모의 난관에 이식합니다.
> >     3.  태어난 새끼의 **10~30%** 정도만 유전자를 가집니다.
> > * **한계점 (P.51):** 유전자가 게놈의 **무작위 위치(Randomly)**에 끼어들어갑니다. 엉뚱한 곳에 들어가서 암을 유발하거나(Tumor), 발현이 안 될 수도 있습니다.

> [!NOTE] Pages 52-55: 유전자 적중 (Gene Targeting / Knockout)
> Transgenic의 한계(무작위 삽입)를 극복하고, 원하는 유전자만 정확히 없애거나 교체하는 기술입니다. **"Homologous Recombination (상동 재조합)"**이 핵심입니다.
>
> > [!IMPORTANT] PAGE 52-54: 선별 과정 (Positive/Negative Selection) **[기전 상세]**
> > ES cell(배아줄기세포)에 유전자를 넣고, 제대로 들어간 놈만 어떻게 골라낼까요? **Neo**와 **TK** 유전자를 씁니다.
> >
> > 1.  **구성:** 타겟 유전자 중간에 **NeoR**(항생제 내성)을 넣고, 끝부분에 **tkHSV**(자살 유전자)를 달아둡니다.
> > 2.  **경우의 수 (P.54):**
> >     * **Case A (Homologous Recombination - 성공):** 타겟 위치에 정확히 교체됨. **NeoR은 들어가고, TK는 떨어져 나감.**
> >     * **Case B (Random Insertion - 실패):** 엉뚱한 곳에 통째로 들어감. **NeoR도 있고, TK도 있음.**
> >     * **Case C (No Insertion):** 아무것도 안 들어감.
> > 3.  **약물 처리 결과:**
> >     * **Neomycin 처리:** NeoR이 없는 Case C 사망.
> > 	* **Ganciclovir 처리:** TK가 있는 Case B 사망 (Ganciclovir가 TK에 의해 독성물질로 변함).
> >     * **최종 생존:** **Case A (성공한 세포)**만 살아남습니다.
>
> > [!INFO] PAGE 55: 키메라(Chimera) 마우스
> > * 유전자가 조작된 ES cell(검은 쥐 유래)을 흰 쥐의 배반포(Blastocyst)에 넣습니다.
> > * 태어난 쥐는 흰색/검은색이 섞인 **키메라**가 됩니다. 이 쥐를 다시 교배해서 온몸이 검은색인(생식세포까지 전달된) 쥐를 선별해야 완전한 Knockout 마우스가 됩니다.

> [!NOTE] Pages 56-63: Cre-Lox 시스템 (Inducible Deletion)
> 유전자를 그냥 없애버리면 태어나자마자 죽을 수도 있습니다(Lethal). 그래서 **"특정 시기, 특정 조직에서만"** 유전자를 없애는 고급 기술이 필요합니다.
>
> > [!IMPORTANT] PAGE 56-59: Cre-LoxP 원리 **[핵심 원리]**
> > * **도구:**
> >     * **Cre Recombinase:** P1 파지에서 유래한 효소. **loxP** 자리를 인식해 그 사이를 잘라냅니다.
> >     * **loxP:** 34bp의 특정 DNA 서열. 타겟 유전자 양옆에 붙여둡니다(Flanked).
> > * **전략 (Tissue-specific Knockout):**
> >     * 쥐 A: **DNA Polymerase $\beta$** 유전자 양옆에 **loxP**를 심어둠 (Floxed mouse).
> >     * 쥐 B: **Thymus-specific promoter** 뒤에 **Cre** 유전자를 달아둠.
> >     * 교배(A x B): 태어난 쥐는 온몸에 loxP가 있지만, 오직 **흉선(Thymus)**에서만 Cre가 나와서 해당 유전자를 잘라버립니다.
> >     * *결과: 다른 조직은 멀쩡하고, 흉선에서만 유전자가 결손됩니다.*
>
> > [!TIP] PAGE 60: Cre-Lox Turn-on (발현 유도)
> > 반대로 유전자를 켤 수도 있습니다.
> > * **전략:** **Promoter - [loxP - STOP - loxP] - Gene**
> > * 평소에는 STOP 코돈 때문에 발현이 안 됩니다.
> > * Cre가 작동하면 **STOP 코돈이 잘려 나가면서(Deletion)** 유전자가 켜집니다. 독성 유전자(Toxic gene) 연구에 씁니다.
>
> > [!IMPORTANT] PAGE 61-62: Cre-ER (Tamoxifen 유도) **[약물 기전]**
> > 조직 특이성에 **시간(Timing)** 조절까지 더한 기술입니다.
> > * **구조:** Cre 효소에 **Estrogen Receptor (ER)**를 붙여놨습니다.
> > * **평소 (Without Tamoxifen):** HSP90이라는 단백질이 Cre-ER를 잡고 있어서 핵 안으로 못 들어갑니다(Inactive).
> > * **약물 처리 (With Tamoxifen):** **Tamoxifen**을 주사하면 ER에 결합하고, HSP90이 떨어져 나갑니다.
> > * **결과:** 자유로워진 Cre-ER이 **핵(Nucleus)**으로 이동해 loxP를 자릅니다.
> > * *이로써 연구자가 원할 때(Time-specific) 유전자를 조작할 수 있게 됩니다.*

> [!abstract] 강의 요약: 면역학 연구기법 I (Flowchart)
>
> 이번 강의는 크게 **"시각화(Seeing) -> 분석(Analyzing) -> 조작(Manipulating)"**의 흐름으로 진행되었습니다.
>
> 1.  **Visualization (단백질과 세포 보기)**
>     * **고정된 조직:** 항체와 효소(HRP)를 이용한 **IHC**로 위치와 문맥 파악.
>     * **세밀한 구조:** 금 입자를 이용한 **IEM**으로 해상도 극대화.
>     * **살아있는 세포:** 형광(Fluorescence)과 **공초점/다광자 현미경**을 이용해 3D 및 실시간(Intravital) 관찰.
>
> 2.  **Genomic Analysis (유전자와 염색질 분석)**
>     * **위치 확인:** **FISH**를 통해 염색체 전좌(예: CML)나 유전자 거리를 시각화.
>     * **상호작용:** **ChIP**으로 단백질-DNA 결합을, **3C**로 DNA-DNA 상호작용을 분석.
>
> 3.  **Genetic Manipulation (유전자 조작 및 모델링)**
>     * **유전자 가위:** **CRISPR-Cas9**을 이용해 특정 유전자를 자르거나(KO), 조절(CRISPRi).
>     * **동물 모델:** **Transgenic** 및 **Knockout** 마우스 생성.
>     * **정밀 제어:** **Cre-Lox 시스템**을 이용해 "원하는 조직에서, 원하는 때에" 유전자를 조작.

> [!example] 핵심 키워드 정리 (Key Terms & Concepts)
>
> ### 1. Imaging & Microscopy
> * **IHC vs ICC:** 조직(Context) vs 세포(Localization).
> * **HRP & DAB:** 갈색(Brown) 발색의 표준. (cf. AP & pNPP는 Yellow).
> * **Stokes Shift:** 흡수 파장보다 방출 파장이 더 길다(에너지 손실).
> * **Fluorophores:** FITC(Green/525nm), PE(Red-Orange/Bright!), APC(Red/660nm).
> * **Confocal vs Multiphoton:**
>     * **Confocal:** **Pinhole**로 초점면 외 빛 차단 -> 선명한 상.
>     * **Multiphoton:** **장파장(적외선)** + 2개 광자 동시 타격 -> **Deep penetration** & **Low photobleaching**.
>
> ### 2. Chromatin & Genomics
> * **FISH:** 3D 구조 및 유전자 거리 측정 (CML의 **BCR-ABL** 전좌 확인).
> * **ChIP (Chromatin IP):** 단백질-DNA 결합 분석. (Enrichment = Input 대비 비율).
> * **Histone Markers:**
>     * Active: **H3K4me3**, Acetylation.
>     * Inactive: **H3K9me3**, **H3K27me3**.
>
> ### 3. CRISPR-Cas9
> * **Components:** Cas9(가위) + **sgRNA**(crRNA+tracrRNA) + **PAM**(5'-NGG-3').
> * **Repair:** **NHEJ**(Indel/KO) vs **HDR**(Correction).
> * **dCas9 (Dead Cas9):** **D10A & H840A** 변이. 자르지 않고 붙기만 함. (CRISPRi, Epigenetic editing 등에 응용).
>
> ### 4. Animal Models
> * **3Rs:** Replace, Reduce, Refine.
> * **Strains:**
>     * **Syngeneic (동계):** 유전자 동일 (Inbred).
>     * **Congenic:** 특정 유전자좌(Locus) 하나만 다름.
> * **Gene Targeting (Knockout):**
>     * **Homologous Recombination** 이용.
>     * **Selection:** **Positive(NeoR)** + **Negative(TK/Ganciclovir)**.
> * **Cre-Lox System (족보):**
>     * **Cre:** 자르는 효소 (Tissue-specific promoter).
>     * **loxP:** 잘리는 위치 (Target gene flanking).
>     * **Inducible:** **Cre-ER + Tamoxifen** (시간 조절 가능).

> >     3.  나만의 면역학적 의문을 해결할 **실험 설계(Experimental Design)** 능력을 길러야 합니다.
>
> > [!INFO] PAGE 3-4: 항체 매개 시각화 (개요)
> > * **주제:** 항체 매개 미세구조 시각화 (Antibody-Mediated Microscopic Visualization).
> > * **두 가지 큰 줄기 (P.4):**
> >     1.  **IHC/ICC:** 효소가 달린 항체(Enzyme-Conjugated Antibodies)를 이용해 **고정된 조직(Fixed Tissues)**의 이미지를 만듭니다.
> >     2.  **Immunoelectron Microscopy (면역전자현미경):** 금 입자(Gold Beads)를 사용하여 항체가 결합한 항원을 전자현미경으로 봅니다.
>
> > [!IMPORTANT] PAGE 5: ICC vs IHC (정의 및 차이)
> > 이 두 용어의 정의를 정확히 구분하세요. 대상(Target)이 다릅니다.
> > * **Immunocytochemistry (ICC, 면역세포화학법):**
> >     * 대상: **Cells** (배양 세포, 현탁액 세포).
> >     * 목적: 세포 내 특정 단백질(항원)의 **위치(Localization)**를 시각화.
> > * **Immunohistochemistry (IHC, 면역조직화학법):**
> >     * 대상: **Tissue section** (조직 절편).
> >     * 목적: 생체 조직(Biological tissues) 내에서 세포들의 분포와 항원의 위치를 **공간적 문맥(Spatial Context)** 속에서 파악.
> >     * *피라미드 그림을 보세요. Cells -> Tissues -> Organs 순으로 올라갑니다. Tissues 레벨을 보는 게 IHC입니다.*
>
> > [!IMPORTANT] PAGE 6: 효소와 기질 (Enzymes & Substrates) **[상세 암기]**
> > 항체 끝에 달린 **효소(Enzyme)**가 **기질(Substrate)**을 만나 **색깔(Color)**을 내는 화학 반응입니다. 표의 내용을 달달 외우세요.
> >
> > **1. HRP (Horseradish Peroxidase) 효소 기질:**
> > * **DAB (3,3'-diaminobenzidine):** **Brown (갈색)**. 가장 대표적입니다.
> > * **TMB (3,3',5,5'-tetramethylbenzidine):** **Blue** (흡광도 650nm). 주로 ELISA에 씁니다.
> > * **ABTS:** **Blue-green** (흡광도 410nm).
> > * **Luminol:** 발광(Light) 반응.
> >
> > **2. AP (Alkaline Phosphatase) 효소 기질:**
> > * **pNPP (p-Nitrophenyl Phosphate):** **Yellow** (흡광도 405nm).
> >
> > **3. Background (위양성 주의):**
> > * **적혈구(RBC):** Peroxidase가 많아서 HRP 사용 시 배경이 지저분해집니다.
> > * **폐포 세포(Alveolar cells):** AP가 많습니다.
> > * **유색 조직(Pigmented tissue):** 흑색종(Melanoma)처럼 원래 까만 조직은 갈색인 DAB를 쓰면 안 보입니다. 이때는 Purple, Red, Blue 계열 기질을 써야 합니다.
>
> > [!INFO] PAGE 7-8: IHC 데이터 해석 및 워크플로우
> > * **데이터 해석(P.7):**
> >     * **그림:** 제1형 당뇨병 모델(NOD mice). 파란색은 Insulin(베타세포), 갈색(Brown-red)은 CD8 T세포입니다.
> >     * **의미:** CD8 T세포가 인슐린을 만드는 베타세포를 공격(Killing)하여 파괴하는 자가면역 기전을 보여줍니다. (PD-L1과 PD-1 상호작용 관련 내용 포함)
> > * **실험 과정(P.8):**
> >     * Fixation(심장 관류) -> Dehydration -> Embedding(Paraffin/OCT) -> Sectioning -> Staining -> Mounting -> Imaging. 이 순서, 머릿속에 그리세요.
>
> > [!TIP] PAGE 9: 임상 판독 기준 (Scoring System) **[기출 포인트]**
> > 병리과에서 암 진단 시 사용하는 **HER2 IHC 판독 기준**입니다. 수치 싸움입니다.
> > * **Score 0 (Negative):** 염색 없음 혹은 **10% 미만**의 종양세포에서 불완전한 막 염색.
> > * **Score 1+ (Negative):** **10% 이상**에서 희미하게 보이나, **불완전한(Incomplete)** 막 염색.
> > * **Score 2+ (Weakly Positive):** **10% 이상**에서 약~중간 강도의 **완전한(Complete)** 막 염색.
> > * **Score 3+ (Positive):** **10% 이상**에서 **강한(Strong)** 완전한 막 염색.
> > * *핵심은 '10% 이상인가?' 그리고 '막이 완전히(Complete) 둘러싸였는가?'입니다.*

>
> > [!INFO] PAGE 10: Immunoelectron Microscopy (IEM)
> > * **원리:** 항체에 색소 대신 **금 입자(Colloidal Gold Particle)**를 붙입니다. 금은 전자밀도(Electron-dense)가 높아 전자현미경에서 **까만 점(Black dots)**으로 보입니다.

> > * 단순히 보는 것을 넘어, 세포 내 분자(DNA 등)의 시각화, 생체 내(Intravital) 관찰까지 확장됩니다.
> > * 그림의 "Black Light Run"처럼 어둠 속에서 특정 부분만 빛나게 하는 원리입니다.
>
> > [!IMPORTANT] PAGE 12-13: 형광 원리 및 Fluorophores 스펙트럼 **[필수 암기]**
> > **1. Jablonski Diagram (P.12):**
> > * 빛을 흡수(Absorbance) -> 들뜬 상태(Excited) -> 에너지 일부 손실 -> 빛 방출(Emission).
> > * **Stokes Shift:** 방출되는 빛(Emission)은 흡수된 빛보다 항상 **파장이 깁니다(Longer wavelength)**. 에너지가 낮아졌으니까요.
> >
> > **2. 주요 형광물질 표 (P.13):** 짝을 지어 외우세요.
> > * **DAPI/Alexa 405:** Abs ~400nm / Em ~420nm (Blue). 핵 염색에 씀.
> > * **FITC (Fluorescein):** Abs **490nm** / Em **525nm** (Green). 가장 기본.
> > * **PE (Phycoerythrin):** Abs 490, 565 / Em **578nm** (Red-Orange). **밝기(Relative Brightness)가 5**로 매우 밝습니다.
> > * **APC (Allophycocyanin):** Abs **650nm** / Em **661nm** (Red).
> > * **Cy5:** Abs 650 / Em 670.
> > * *실험 설계 시 레이저 파장과 형광물질의 Absorbance가 맞는지 반드시 확인해야 합니다.*
>
> > [!INFO] PAGE 14: 형광현미경 필터 구조 (Filter Sets)
> > 원하는 파장만 골라내는 **3단계 필터링**이 핵심입니다.
> > 1.  **First barrier filter:** 광원에서 온 빛 중 특정 파장(예: Blue 450-490nm)만 통과시켜 시료를 때립니다(Excitation).
> > 2.  **Beam-splitting mirror:** 510nm 이하는 반사(Reflects), 그 이상은 통과(Transmits). 파장에 따라 길을 나눕니다.
> > 3.  **Second barrier filter:** 시료에서 나온 형광(예: Green 520-560nm)만 통과시키고, 나머지 잡광은 차단(Cuts out)하여 눈(Eyepiece)으로 보냅니다.
>
> > [!TIP] PAGE 15-16: GFP & RFP 그리고 Transgenic Animals
> > * **역사:** 1962년 Shimomura가 해파리(*Aequorea victoria*)에서 **GFP(Green)**를, 1999년 Lukyanov가 산호(Coral)에서 **RFP(Red)**를 발견했습니다. (2008년 노벨 화학상)
> > * **응용(Whole Animal):** 16페이지 사진을 보세요. **Actin promoter** 뒤에 GFP 유전자를 붙였습니다. Actin은 모든 세포에 있으니, 갓 태어난 쥐(Neonatal mice) 전체가 초록색으로 빛납니다.

>
> > [!IMPORTANT] PAGE 17: 공초점 현미경 (Confocal Microscopy)
> > * **핵심 부품:** **Pinhole (핀홀)**.
> > * **기전:** 초점면(Focal plane)에서 출발한 빛만 통과시키고, 초점면 위아래(Out of focus)에서 산란된 빛은 핀홀 주변에서 **차단**됩니다.
> > * **결과:** 흐릿함이 없는 **선명한 상**을 얻고, Z축 스캔을 통해 **3차원 영상** 제작이 가능합니다.
>
> > [!IMPORTANT] PAGE 18: 다광자 현미경 (Multiphoton Microscopy)
> > 공초점의 업그레이드 버전입니다.
> > * **원리:** **장파장/저에너지(Long wavelength, Low energy)**의 적외선 레이저를 씁니다.
> > * **Two-photon Excitation:** 광자 1개로는 에너지가 부족해서, 광자 2개가 동시에 때려야만 형광이 나옵니다. 즉, 초점이 맺힌 점(Focus spot) 외에는 아예 형광이 안 생깁니다. (핀홀 불필요)
> > * **장점:**
> >     1.  **광퇴색(Photobleaching) 최소화:** 에너지가 낮아 검체 손상이 적습니다.
> >     2.  **깊은 투과력:** 장파장은 조직 깊숙이 들어갈 수 있습니다.
>
> > [!TIP] PAGE 19-21: 생체 내 영상 (Intravital Imaging) **[데이터 해석]**
> > 살아있는 동물에서 면역세포의 움직임을 실시간으로 봅니다.
> >
> > **1. 혈관 투과성 실험 (P.19):**
> > * **설계:** 귀 진피(Ear dermis)에 **VEGFA164**(혈관내피성장인자) 주입. 혈관 내에는 **2000 kDa FITC-Dextran**(녹색) 주입.
> > * **결과:** VEGFA 주입 후 혈관 직경이 커지고, 녹색 형광이 혈관 밖으로 새어 나가는 **Vascular leakage**가 관찰됨.
> >
> > **2. 호중구의 박테리아 제거 (P.20):**
> > * **상황:** UPEC(대장균) 감염 모델.
> > * **관찰:** 0분부터 **150분**까지 촬영. 호중구(Neutrophil)가 몰려와 박테리아(형광)를 포식하여 제거(Clearance)하는 과정을 시각화.
> >
> > **3. 림프절 T세포 이동 (P.21):**
> > * **표지:** T세포는 **CFSE (Green)**로, 고내피세동맥(HEV)은 **Red**로 염색.
> > * **결과:** T세포가 HEV를 통해 림프절로 들어와 이동하는 3차원 궤적(Trafficking)을 확인.

> >     * **병리:** 이 융합 단백질은 **Tyrosine Kinase 활성**이 비정상적으로 높아져 암(Tumor)을 유발합니다. (이걸 표적으로 한 약이 글리벡입니다.)

> > [!INFO] PAGE 26-27: 염색질 구조 (Chromatin Structure)
> > * **구조:** DNA는 히스톤(Histone)을 감아 **뉴클레오솜(Nucleosome)**을 형성합니다. (Beads on a string)
> > * **히스톤 꼬리 변형 (Histone Tail Modification):**
> >     * **Acetylation (아세틸화):** 히스톤과 DNA 결합을 느슨하게 함 -> **Open Chromatin** -> **Gene Active**. (전사 기구가 접근 가능)
> >     * **Methylation (메틸화):** 주로 **Closed Chromatin** -> **Gene Inactive**. (단, 위치에 따라 다름)

> >
> > **1. Workflow (P.29):**
> >     * **Cross-linking:** 포름알데히드로 단백질과 DNA를 딱 붙임(Fixation).
> >     * **Fragmentation:** DNA를 잘게 조각냄.
> >     * **Immunoprecipitation (IP):** 특정 단백질에 대한 **항체(Antibody)**로 낚아챔.
> >     * **DNA Extraction & Analysis:** 단백질을 녹여 없애고 DNA만 남겨서 PCR이나 Sequencing으로 분석.
> >     * *결과값: Enrichment = X/R (Input 대비 얼마나 농축됐는가)*
> >
> > **2. 주요 항체 리스트 (P.30 표):** *이거 구분 못하면 해석 못 합니다.*
> >     * **Active Markers (전사 활성):** H3K4me3 (Trimethyl), H3K9ac, H3K14ac (Acetylation은 주로 활성).
> >     * **Repressive Markers (전사 억제):** **H3K9me3**, **H3K27me3**. (꽉 뭉쳐서 유전자 꺼짐)
>
> > [!INFO] PAGE 31-32: 3C (Chromosome Conformation Capture)
> > 멀리 떨어진 DNA끼리의 상호작용(Looping 등)을 봅니다.
> > * **핵심 과정:** Cross-link -> **Restriction Enzyme (제한효소) 절단** -> **Ligation (연결)**.
> > * *포인트: 원래 멀리 있던 DNA라도, 단백질에 의해 묶여서 가까이 있다면 Ligation 되어 하나의 가닥으로 연결됩니다.*
> > * 결과물: Ligation products를 PCR/Sequencing하여 "아, 1번 염색체의 Promoter가 멀리 있는 Enhancer랑 붙어먹었구나"를 알아냅니다.

> [!NOTE] Pages 33-41: 유전자 가위 (CRISPR-Cas9)
> 2020년 노벨 화학상의 주인공입니다. 단순한 가위질을 넘어 유전자 조절까지 확장된 최신 경향을 파악하세요.
>
> > [!INFO] PAGE 33-35: 역사 (Timeline)
> > * **1987:** 대장균(E. coli)에서 CRISPR 서열(Clustered repeats) 최초 발견 (이땐 뭔지 몰랐음).
> > * **2012:** **Doudna & Charpentier** (노벨상 주역)가 Cas9의 DNA 절단 기전 규명.
> > * **2013:** 인간 세포(Mammalian cells)에서 게놈 교정 성공.
> > * **2020:** 노벨 화학상 수상.
>
> > [!IMPORTANT] PAGE 36-37: 작동 기전 (Mechanism) **[핵심 원리]**
> > 박테리아의 적응 면역 시스템에서 유래했습니다.
> >
> > **1. 구성 요소:**
> >     * **Cas9 Nuclease:** 가위(Enzyme).
> >     * **sgRNA (single guide RNA):** **crRNA** (Target 인식 20nt) + **tracrRNA** (Cas9 결합 구조)를 하나로 합친 것.
> >     * **PAM (Protospacer Adjacent Motif):** **5'-NGG-3'**. Cas9이 DNA를 자르려면 타겟 서열 바로 뒤에 무조건 이 NGG가 있어야 합니다. (없으면 못 자름!)
> >
> > **2. 수선 기전 (Repair Pathways):** 잘린 DNA(DSB)는 두 가지로 수선됩니다.
> >     * **NHEJ (Non-homologous end joining):** 대충 갖다 붙임 -> 돌연변이(Indel) 발생 -> **Gene Knockout (유전자 파괴)**.
> >     * **HDR (Homology-directed repair):** 정상 DNA 조각(Donor DNA)을 넣어줌 -> 그걸 보고 베껴서 수선 -> **Gene Correction (유전자 교정/삽입)**.
>
> > [!IMPORTANT] PAGE 38-39: Cas9 구조와 변형 (dCas9) **[상세 구조]**
> > **1. Cas9의 도메인:**
> >     * **RuvC Domain:** 비표적 가닥(Non-target strand) 절단.
> >     * **HNH Domain:** 표적 가닥(Target strand) 절단.
> >     * 이 두 도메인이 모두 작동해야 이중 나선이 잘립니다(DSB).
> >
> > **2. dCas9 (dead Cas9): 가위 기능을 죽인 버전**
> >     * **변이(Mutation):** **D10A** (RuvC 불활성) + **H840A** (HNH 불활성).
> >     * **기능:** DNA를 자르지는 못하고, 가이드 RNA를 따라 **붙기만 합니다(Binding only)**.
> >     * **Nickase:** 둘 중 하나만 죽이면(예: D10A만), 한 가닥만 자르는 Nickase가 됩니다.
>
> > [!TIP] PAGE 40: dCas9의 응용 (CRISPRi & Epigenetics)
> > 자르지 못하는 dCas9에 다른 효소(Effector)를 달아서 씁니다.
> > * **CRISPRi (Interference):** dCas9이 프로모터에 딱 붙어서 전사 기구가 못 오게 막음 -> **Transcription OFF**.
> > * **Epigenetic Editing:**
> >     * **dCas9 + EZH2:** EZH2는 **H3K27 Methyltransferase**입니다. 타겟 유전자의 히스톤을 메틸화시켜 영구적으로 꺼버립니다(Silencing).
> >     * **dCas9 + GFP:** 특정 유전자 위치를 형광으로 추적(Live imaging).

> [!NOTE] Pages 42-51: 동물 실험 기초 및 형질전환 동물
> "왜 쥐를 쓰는가?"부터 "어떻게 유전자를 넣는가?"까지, 동물 실험의 A to Z입니다.
>
> > [!INFO] PAGE 42-43: 동물 실험 윤리 (3Rs)
> > 연구자의 기본 소양입니다. 무조건 외우세요.
> > * **Reduce:** 사용되는 동물의 **수(Number)**를 줄입니다.
> > * **Refine:** 동물에게 가해지는 **고통/스트레스(Stress)**를 최소화하도록 실험법을 개선합니다.
> > * **Replace:** 동물 실험을 세포 실험이나 시뮬레이션 등 **다른 방법(Other methods)**으로 대체합니다.
>
> > [!IMPORTANT] PAGE 44-45: 근교계 (Inbred Strains) **[필수 암기]**
> > 실험 편차를 줄이기 위해 유전자가 99% 이상 동일한 "순계" 마우스를 씁니다.
> > * **생성법:** **형매교배(Brother-sister mating)**를 **20세대** 이상 반복합니다.
> > * **주요 Strain 특성 (P.45 표):** *이거 족보입니다.*
> >     * **BALB/c:** 방사선 감수성(Sensitivity), 하이브리도마(Hybridoma) 제조에 사용.
> >     * **C57BL/6:** 방사선 조사 후 간암(Hepatoma) 발생 빈도 높음, 높은 보체(Complement) 활성.
> >     * **C3H/HeJ:** 유선 종양(Mammary tumor) 발생 빈도 높음.
> >     * **AKR/J:** 백혈병(Leukemia) 발생 빈도 높음.
> >     * **CBA/N:** X-linked 면역결핍(Immunodeficiency).
>
> > [!INFO] PAGE 46-47: Congenic Strains & Adoptive Transfer
> > * **Congenic (동계):** 딱 **하나의 유전자좌(Locus)**만 다르고 나머지 모든 유전자가 동일한 쥐입니다. 특정 유전자의 기능만 콕 집어 연구할 때 씁니다.
> > * **Adoptive Transfer (양자 전달):** 같은 Inbred strain끼리는 거부 반응이 없죠? 그래서 한 쥐의 림프구(Lymphocyte)를 분리해서 다른 쥐에게 주사하여 면역 반응을 전달하는 실험입니다.
>
> > [!IMPORTANT] PAGE 48-51: Transgenic Animals (형질전환) 생성법
> > 외부 유전자(Transgene)를 인위적으로 삽입하는 기술입니다.
> > * **핵심 과정 (Microinjection):**
> >     1.  수정란(Fertilized egg)의 **전핵(Pronucleus)**에 DNA를 직접 찔러 넣습니다.
> >     2.  이 수정란을 **가임신(Pseudopregnant)** 대리모의 난관에 이식합니다.
> >     3.  태어난 새끼의 **10~30%** 정도만 유전자를 가집니다.
> > * **한계점 (P.51):** 유전자가 게놈의 **무작위 위치(Randomly)**에 끼어들어갑니다. 엉뚱한 곳에 들어가서 암을 유발하거나(Tumor), 발현이 안 될 수도 있습니다.

> [!NOTE] Pages 52-55: 유전자 적중 (Gene Targeting / Knockout)
> Transgenic의 한계(무작위 삽입)를 극복하고, 원하는 유전자만 정확히 없애거나 교체하는 기술입니다. **"Homologous Recombination (상동 재조합)"**이 핵심입니다.
>
> > [!IMPORTANT] PAGE 52-54: 선별 과정 (Positive/Negative Selection) **[기전 상세]**
> > ES cell(배아줄기세포)에 유전자를 넣고, 제대로 들어간 놈만 어떻게 골라낼까요? **Neo**와 **TK** 유전자를 씁니다.
> >
> > 1.  **구성:** 타겟 유전자 중간에 **NeoR**(항생제 내성)을 넣고, 끝부분에 **tkHSV**(자살 유전자)를 달아둡니다.
> > 2.  **경우의 수 (P.54):**
> >     * **Case A (Homologous Recombination - 성공):** 타겟 위치에 정확히 교체됨. **NeoR은 들어가고, TK는 떨어져 나감.**
> >     * **Case B (Random Insertion - 실패):** 엉뚱한 곳에 통째로 들어감. **NeoR도 있고, TK도 있음.**
> >     * **Case C (No Insertion):** 아무것도 안 들어감.
> > 3.  **약물 처리 결과:**
> >     * **Neomycin 처리:** NeoR이 없는 Case C 사망.
> > 	* **Ganciclovir 처리:** TK가 있는 Case B 사망 (Ganciclovir가 TK에 의해 독성물질로 변함).
> >     * **최종 생존:** **Case A (성공한 세포)**만 살아남습니다.
>
> > [!INFO] PAGE 55: 키메라(Chimera) 마우스
> > * 유전자가 조작된 ES cell(검은 쥐 유래)을 흰 쥐의 배반포(Blastocyst)에 넣습니다.
> > * 태어난 쥐는 흰색/검은색이 섞인 **키메라**가 됩니다. 이 쥐를 다시 교배해서 온몸이 검은색인(생식세포까지 전달된) 쥐를 선별해야 완전한 Knockout 마우스가 됩니다.

> [!NOTE] Pages 56-63: Cre-Lox 시스템 (Inducible Deletion)
> 유전자를 그냥 없애버리면 태어나자마자 죽을 수도 있습니다(Lethal). 그래서 **"특정 시기, 특정 조직에서만"** 유전자를 없애는 고급 기술이 필요합니다.
>
> > [!IMPORTANT] PAGE 56-59: Cre-LoxP 원리 **[핵심 원리]**
> > * **도구:**
> >     * **Cre Recombinase:** P1 파지에서 유래한 효소. **loxP** 자리를 인식해 그 사이를 잘라냅니다.
> >     * **loxP:** 34bp의 특정 DNA 서열. 타겟 유전자 양옆에 붙여둡니다(Flanked).
> > * **전략 (Tissue-specific Knockout):**
> >     * 쥐 A: **DNA Polymerase $\beta$** 유전자 양옆에 **loxP**를 심어둠 (Floxed mouse).
> >     * 쥐 B: **Thymus-specific promoter** 뒤에 **Cre** 유전자를 달아둠.
> >     * 교배(A x B): 태어난 쥐는 온몸에 loxP가 있지만, 오직 **흉선(Thymus)**에서만 Cre가 나와서 해당 유전자를 잘라버립니다.
> >     * *결과: 다른 조직은 멀쩡하고, 흉선에서만 유전자가 결손됩니다.*
>
> > [!TIP] PAGE 60: Cre-Lox Turn-on (발현 유도)
> > 반대로 유전자를 켤 수도 있습니다.
> > * **전략:** **Promoter - [loxP - STOP - loxP] - Gene**
> > * 평소에는 STOP 코돈 때문에 발현이 안 됩니다.
> > * Cre가 작동하면 **STOP 코돈이 잘려 나가면서(Deletion)** 유전자가 켜집니다. 독성 유전자(Toxic gene) 연구에 씁니다.
>
> > [!IMPORTANT] PAGE 61-62: Cre-ER (Tamoxifen 유도) **[약물 기전]**
> > 조직 특이성에 **시간(Timing)** 조절까지 더한 기술입니다.
> > * **구조:** Cre 효소에 **Estrogen Receptor (ER)**를 붙여놨습니다.
> > * **평소 (Without Tamoxifen):** HSP90이라는 단백질이 Cre-ER를 잡고 있어서 핵 안으로 못 들어갑니다(Inactive).
> > * **약물 처리 (With Tamoxifen):** **Tamoxifen**을 주사하면 ER에 결합하고, HSP90이 떨어져 나갑니다.
> > * **결과:** 자유로워진 Cre-ER이 **핵(Nucleus)**으로 이동해 loxP를 자릅니다.
> > * *이로써 연구자가 원할 때(Time-specific) 유전자를 조작할 수 있게 됩니다.*

> [!abstract] 강의 요약: 면역학 연구기법 I (Flowchart)
>
> 이번 강의는 크게 **"시각화(Seeing) -> 분석(Analyzing) -> 조작(Manipulating)"**의 흐름으로 진행되었습니다.
>
> 1.  **Visualization (단백질과 세포 보기)**
>     * **고정된 조직:** 항체와 효소(HRP)를 이용한 **IHC**로 위치와 문맥 파악.
>     * **세밀한 구조:** 금 입자를 이용한 **IEM**으로 해상도 극대화.
>     * **살아있는 세포:** 형광(Fluorescence)과 **공초점/다광자 현미경**을 이용해 3D 및 실시간(Intravital) 관찰.
>
> 2.  **Genomic Analysis (유전자와 염색질 분석)**
>     * **위치 확인:** **FISH**를 통해 염색체 전좌(예: CML)나 유전자 거리를 시각화.
>     * **상호작용:** **ChIP**으로 단백질-DNA 결합을, **3C**로 DNA-DNA 상호작용을 분석.
>
> 3.  **Genetic Manipulation (유전자 조작 및 모델링)**
>     * **유전자 가위:** **CRISPR-Cas9**을 이용해 특정 유전자를 자르거나(KO), 조절(CRISPRi).
>     * **동물 모델:** **Transgenic** 및 **Knockout** 마우스 생성.
>     * **정밀 제어:** **Cre-Lox 시스템**을 이용해 "원하는 조직에서, 원하는 때에" 유전자를 조작.

> [!example] 핵심 키워드 정리 (Key Terms & Concepts)
>
> ### 1. Imaging & Microscopy
> * **IHC vs ICC:** 조직(Context) vs 세포(Localization).
> * **HRP & DAB:** 갈색(Brown) 발색의 표준. (cf. AP & pNPP는 Yellow).
> * **Stokes Shift:** 흡수 파장보다 방출 파장이 더 길다(에너지 손실).
> * **Fluorophores:** FITC(Green/525nm), PE(Red-Orange/Bright!), APC(Red/660nm).
> * **Confocal vs Multiphoton:**
>     * **Confocal:** **Pinhole**로 초점면 외 빛 차단 -> 선명한 상.
>     * **Multiphoton:** **장파장(적외선)** + 2개 광자 동시 타격 -> **Deep penetration** & **Low photobleaching**.
>
> ### 2. Chromatin & Genomics
> * **FISH:** 3D 구조 및 유전자 거리 측정 (CML의 **BCR-ABL** 전좌 확인).
> * **ChIP (Chromatin IP):** 단백질-DNA 결합 분석. (Enrichment = Input 대비 비율).
> * **Histone Markers:**
>     * Active: **H3K4me3**, Acetylation.
>     * Inactive: **H3K9me3**, **H3K27me3**.
>
> ### 3. CRISPR-Cas9
> * **Components:** Cas9(가위) + **sgRNA**(crRNA+tracrRNA) + **PAM**(5'-NGG-3').
> * **Repair:** **NHEJ**(Indel/KO) vs **HDR**(Correction).
> * **dCas9 (Dead Cas9):** **D10A & H840A** 변이. 자르지 않고 붙기만 함. (CRISPRi, Epigenetic editing 등에 응용).
>
> ### 4. Animal Models
> * **3Rs:** Replace, Reduce, Refine.
> * **Strains:**
>     * **Syngeneic (동계):** 유전자 동일 (Inbred).
>     * **Congenic:** 특정 유전자좌(Locus) 하나만 다름.
> * **Gene Targeting (Knockout):**
>     * **Homologous Recombination** 이용.
>     * **Selection:** **Positive(NeoR)** + **Negative(TK/Ganciclovir)**.
> * **Cre-Lox System (족보):**
>     * **Cre:** 자르는 효소 (Tissue-specific promoter).
>     * **loxP:** 잘리는 위치 (Target gene flanking).
>     * **Inducible:** **Cre-ER + Tamoxifen** (시간 조절 가능).

> [!NOTE] Pages 1-9: 항체 매개 시각화 기초 (IHC/ICC) 및 발색 원리
> 면역학 연구기법의 첫 단추는 **"보이지 않는 단백질을 항체로 염색해서 눈으로 보는 것"**입니다. 가장 기초적이면서도 임상/연구 모두에서 필수적인 파트입니다.
>
> > [!INFO] PAGE 1-2: 강의 목표 및 개요
> > * **표지 이미지(P.1):** 공초점 현미경 등으로 촬영된 세포들의 화려한 이미지가 보입니다. 앞으로 배울 기술들이죠.
> > * **핵심 목표(Goals, P.2):** 단순히 실험 방법만 외우는 게 아닙니다.
> >     1.  면역학 관련 문헌을 읽고 **방법론의 원리**를 이해해야 합니다.
> >     2.  연구자가 선택한 실험 기법이 **"적절한지"** 판단할 수 있는 비판적 사고가 필요합니다.
> >     3.  나만의 면역학적 의문을 해결할 **실험 설계(Experimental Design)** 능력을 길러야 합니다.
>
> > [!INFO] PAGE 3-4: 항체 매개 시각화 (개요)
> > * **주제:** 항체 매개 미세구조 시각화 (Antibody-Mediated Microscopic Visualization).
> > * **두 가지 큰 줄기 (P.4):**
> >     1.  **IHC/ICC:** 효소가 달린 항체(Enzyme-Conjugated Antibodies)를 이용해 **고정된 조직(Fixed Tissues)**의 이미지를 만듭니다.
> >     2.  **Immunoelectron Microscopy (면역전자현미경):** 금 입자(Gold Beads)를 사용하여 항체가 결합한 항원을 전자현미경으로 봅니다.
>
> > [!IMPORTANT] PAGE 5: ICC vs IHC (정의 및 차이)
> > 이 두 용어의 정의를 정확히 구분하세요. 대상(Target)이 다릅니다.
> > * **Immunocytochemistry (ICC, 면역세포화학법):**
> >     * 대상: **Cells** (배양 세포, 현탁액 세포).
> >     * 목적: 세포 내 특정 단백질(항원)의 **위치(Localization)**를 시각화.
> > * **Immunohistochemistry (IHC, 면역조직화학법):**
> >     * 대상: **Tissue section** (조직 절편).
> >     * 목적: 생체 조직(Biological tissues) 내에서 세포들의 분포와 항원의 위치를 **공간적 문맥(Spatial Context)** 속에서 파악.
> >     * *피라미드 그림을 보세요. Cells -> Tissues -> Organs 순으로 올라갑니다. Tissues 레벨을 보는 게 IHC입니다.*
>
> > [!IMPORTANT] PAGE 6: 효소와 기질 (Enzymes & Substrates) **[상세 암기]**
> > 항체 끝에 달린 **효소(Enzyme)**가 **기질(Substrate)**을 만나 **색깔(Color)**을 내는 화학 반응입니다. 표의 내용을 달달 외우세요.
> >
> > **1. HRP (Horseradish Peroxidase) 효소 기질:**
> > * **DAB (3,3'-diaminobenzidine):** **Brown (갈색)**. 가장 대표적입니다.
> > * **TMB (3,3',5,5'-tetramethylbenzidine):** **Blue** (흡광도 650nm). 주로 ELISA에 씁니다.
> > * **ABTS:** **Blue-green** (흡광도 410nm).
> > * **Luminol:** 발광(Light) 반응.
> >
> > **2. AP (Alkaline Phosphatase) 효소 기질:**
> > * **pNPP (p-Nitrophenyl Phosphate):** **Yellow** (흡광도 405nm).
> >
> > **3. Background (위양성 주의):**
> > * **적혈구(RBC):** Peroxidase가 많아서 HRP 사용 시 배경이 지저분해집니다.
> > * **폐포 세포(Alveolar cells):** AP가 많습니다.
> > * **유색 조직(Pigmented tissue):** 흑색종(Melanoma)처럼 원래 까만 조직은 갈색인 DAB를 쓰면 안 보입니다. 이때는 Purple, Red, Blue 계열 기질을 써야 합니다.
>
> > [!INFO] PAGE 7-8: IHC 데이터 해석 및 워크플로우
> > * **데이터 해석(P.7):**
> >     * **그림:** 제1형 당뇨병 모델(NOD mice). 파란색은 Insulin(베타세포), 갈색(Brown-red)은 CD8 T세포입니다.
> >     * **의미:** CD8 T세포가 인슐린을 만드는 베타세포를 공격(Killing)하여 파괴하는 자가면역 기전을 보여줍니다. (PD-L1과 PD-1 상호작용 관련 내용 포함)
> > * **실험 과정(P.8):**
> >     * Fixation(심장 관류) -> Dehydration -> Embedding(Paraffin/OCT) -> Sectioning -> Staining -> Mounting -> Imaging. 이 순서, 머릿속에 그리세요.
>
> > [!TIP] PAGE 9: 임상 판독 기준 (Scoring System) **[기출 포인트]**
> > 병리과에서 암 진단 시 사용하는 **HER2 IHC 판독 기준**입니다. 수치 싸움입니다.
> > * **Score 0 (Negative):** 염색 없음 혹은 **10% 미만**의 종양세포에서 불완전한 막 염색.
> > * **Score 1+ (Negative):** **10% 이상**에서 희미하게 보이나, **불완전한(Incomplete)** 막 염색.
> > * **Score 2+ (Weakly Positive):** **10% 이상**에서 약~중간 강도의 **완전한(Complete)** 막 염색.
> > * **Score 3+ (Positive):** **10% 이상**에서 **강한(Strong)** 완전한 막 염색.
> > * *핵심은 '10% 이상인가?' 그리고 '막이 완전히(Complete) 둘러싸였는가?'입니다.*

> [!NOTE] Pages 10-16: 면역전자현미경(IEM) & 형광 시각화
> 해상도(Resolution)의 끝판왕인 전자현미경과, 특정 분자를 빛나게 하는 형광 기법입니다.
>
> > [!INFO] PAGE 10: Immunoelectron Microscopy (IEM)
> > * **원리:** 항체에 색소 대신 **금 입자(Colloidal Gold Particle)**를 붙입니다. 금은 전자밀도(Electron-dense)가 높아 전자현미경에서 **까만 점(Black dots)**으로 보입니다.
> > * **다중 표지(Double Labeling):** 금 입자의 **크기(Size)**를 다르게 하여 두 가지 항원을 동시에 봅니다.
> >     * 예시 데이터: **MHC class II는 30nm**, **Class I은 15nm** 금 입자로 표지하여 세포 내 위치 관계를 분석했습니다.
>
> > [!INFO] PAGE 11: 면역형광(IF) 개요
> > * 단순히 보는 것을 넘어, 세포 내 분자(DNA 등)의 시각화, 생체 내(Intravital) 관찰까지 확장됩니다.
> > * 그림의 "Black Light Run"처럼 어둠 속에서 특정 부분만 빛나게 하는 원리입니다.
>
> > [!IMPORTANT] PAGE 12-13: 형광 원리 및 Fluorophores 스펙트럼 **[필수 암기]**
> > **1. Jablonski Diagram (P.12):**
> > * 빛을 흡수(Absorbance) -> 들뜬 상태(Excited) -> 에너지 일부 손실 -> 빛 방출(Emission).
> > * **Stokes Shift:** 방출되는 빛(Emission)은 흡수된 빛보다 항상 **파장이 깁니다(Longer wavelength)**. 에너지가 낮아졌으니까요.
> >
> > **2. 주요 형광물질 표 (P.13):** 짝을 지어 외우세요.
> > * **DAPI/Alexa 405:** Abs ~400nm / Em ~420nm (Blue). 핵 염색에 씀.
> > * **FITC (Fluorescein):** Abs **490nm** / Em **525nm** (Green). 가장 기본.
> > * **PE (Phycoerythrin):** Abs 490, 565 / Em **578nm** (Red-Orange). **밝기(Relative Brightness)가 5**로 매우 밝습니다.
> > * **APC (Allophycocyanin):** Abs **650nm** / Em **661nm** (Red).
> > * **Cy5:** Abs 650 / Em 670.
> > * *실험 설계 시 레이저 파장과 형광물질의 Absorbance가 맞는지 반드시 확인해야 합니다.*
>
> > [!INFO] PAGE 14: 형광현미경 필터 구조 (Filter Sets)
> > 원하는 파장만 골라내는 **3단계 필터링**이 핵심입니다.
> > 1.  **First barrier filter:** 광원에서 온 빛 중 특정 파장(예: Blue 450-490nm)만 통과시켜 시료를 때립니다(Excitation).
> > 2.  **Beam-splitting mirror:** 510nm 이하는 반사(Reflects), 그 이상은 통과(Transmits). 파장에 따라 길을 나눕니다.
> > 3.  **Second barrier filter:** 시료에서 나온 형광(예: Green 520-560nm)만 통과시키고, 나머지 잡광은 차단(Cuts out)하여 눈(Eyepiece)으로 보냅니다.
>
> > [!TIP] PAGE 15-16: GFP & RFP 그리고 Transgenic Animals
> > * **역사:** 1962년 Shimomura가 해파리(*Aequorea victoria*)에서 **GFP(Green)**를, 1999년 Lukyanov가 산호(Coral)에서 **RFP(Red)**를 발견했습니다. (2008년 노벨 화학상)
> > * **응용(Whole Animal):** 16페이지 사진을 보세요. **Actin promoter** 뒤에 GFP 유전자를 붙였습니다. Actin은 모든 세포에 있으니, 갓 태어난 쥐(Neonatal mice) 전체가 초록색으로 빛납니다.

> [!NOTE] Pages 17-21: 고급 이미징 (Confocal, Multiphoton, Intravital)
> 이제 **3차원(3D)**과 **살아있는(Live)** 상태를 보는 첨단 기법입니다.
>
> > [!IMPORTANT] PAGE 17: 공초점 현미경 (Confocal Microscopy)
> > * **핵심 부품:** **Pinhole (핀홀)**.
> > * **기전:** 초점면(Focal plane)에서 출발한 빛만 통과시키고, 초점면 위아래(Out of focus)에서 산란된 빛은 핀홀 주변에서 **차단**됩니다.
> > * **결과:** 흐릿함이 없는 **선명한 상**을 얻고, Z축 스캔을 통해 **3차원 영상** 제작이 가능합니다.
>
> > [!IMPORTANT] PAGE 18: 다광자 현미경 (Multiphoton Microscopy)
> > 공초점의 업그레이드 버전입니다.
> > * **원리:** **장파장/저에너지(Long wavelength, Low energy)**의 적외선 레이저를 씁니다.
> > * **Two-photon Excitation:** 광자 1개로는 에너지가 부족해서, 광자 2개가 동시에 때려야만 형광이 나옵니다. 즉, 초점이 맺힌 점(Focus spot) 외에는 아예 형광이 안 생깁니다. (핀홀 불필요)
> > * **장점:**
> >     1.  **광퇴색(Photobleaching) 최소화:** 에너지가 낮아 검체 손상이 적습니다.
> >     2.  **깊은 투과력:** 장파장은 조직 깊숙이 들어갈 수 있습니다.
>
> > [!TIP] PAGE 19-21: 생체 내 영상 (Intravital Imaging) **[데이터 해석]**
> > 살아있는 동물에서 면역세포의 움직임을 실시간으로 봅니다.
> >
> > **1. 혈관 투과성 실험 (P.19):**
> > * **설계:** 귀 진피(Ear dermis)에 **VEGFA164**(혈관내피성장인자) 주입. 혈관 내에는 **2000 kDa FITC-Dextran**(녹색) 주입.
> > * **결과:** VEGFA 주입 후 혈관 직경이 커지고, 녹색 형광이 혈관 밖으로 새어 나가는 **Vascular leakage**가 관찰됨.
> >
> > **2. 호중구의 박테리아 제거 (P.20):**
> > * **상황:** UPEC(대장균) 감염 모델.
> > * **관찰:** 0분부터 **150분**까지 촬영. 호중구(Neutrophil)가 몰려와 박테리아(형광)를 포식하여 제거(Clearance)하는 과정을 시각화.
> >
> > **3. 림프절 T세포 이동 (P.21):**
> > * **표지:** T세포는 **CFSE (Green)**로, 고내피세동맥(HEV)은 **Red**로 염색.
> > * **결과:** T세포가 HEV를 통해 림프절로 들어와 이동하는 3차원 궤적(Trafficking)을 확인.

> 유전자의 위치를 눈으로 확인하고, 유전자가 감겨 있는 히스톤의 상태를 분석하는 파트입니다.
>
> > [!INFO] PAGE 22-23: 3-D FISH (Fluorescence in situ hybridization)
> > * **정의:** 형광 RNA 탐침(Probe)을 DNA에 붙여서 염색체의 3차원적 구조와 거리를 측정합니다.
> > * **거리 측정 데이터 (P.23):**
> >     * **T cells:** $V_H$ (Red)와 $V_{H}J558$ (Green) 유전자 거리가 멉니다(Ready discriminated). 아직 재조합 전이라 그렇습니다.
> >     * **Pro-B cells:** 면역글로불린 유전자좌(Ig locus)가 **수축(Contraction)**합니다. 두 유전자(Red/Green)가 가까워져서 색이 겹쳐 보일 정도입니다. 이게 유전자 재조합의 전조 현상입니다.
>
> > [!IMPORTANT] PAGE 24: 만성 골수성 백혈병 (CML)과 필라델피아 염색체 **[병리 연계]**
> > 이 그림은 족보 중의 족보입니다.
> > * **정상:** 9번(Green)과 22번(Red) 염색체는 떨어져 있어야 합니다.
> > * **CML (Chronic Myeloid Leukemia):**
> >     * 9번과 22번 사이에 **상호 전좌(Reciprocal Translocation)**가 일어납니다.
> >     * **결과:** 9번의 **ABL** 유전자와 22번의 **BCR** 유전자가 합쳐져 **BCR-ABL** 융합 유전자가 생깁니다.
> >     * **병리:** 이 융합 단백질은 **Tyrosine Kinase 활성**이 비정상적으로 높아져 암(Tumor)을 유발합니다. (이걸 표적으로 한 약이 글리벡입니다.)
> >     * *그림 B/C를 보면 Red/Green 점이 붙어서 노란색/중간색으로 보이는 게 전좌의 증거입니다.*
>
> > [!INFO] PAGE 26-27: 염색질 구조 (Chromatin Structure)
> > * **구조:** DNA는 히스톤(Histone)을 감아 **뉴클레오솜(Nucleosome)**을 형성합니다. (Beads on a string)
> > * **히스톤 꼬리 변형 (Histone Tail Modification):**
> >     * **Acetylation (아세틸화):** 히스톤과 DNA 결합을 느슨하게 함 -> **Open Chromatin** -> **Gene Active**. (전사 기구가 접근 가능)
> >     * **Methylation (메틸화):** 주로 **Closed Chromatin** -> **Gene Inactive**. (단, 위치에 따라 다름)

> [!NOTE] Pages 28-32: 단백질-DNA 상호작용 분석 (ChIP & 3C)
> "이 단백질이 DNA 어디에 붙었니?"(ChIP)와 "이 DNA가 저 DNA랑 만났니?"(3C)를 묻는 기술입니다.
>
> > [!IMPORTANT] PAGE 28-30: ChIP (Chromatin Immunoprecipitation) **[실험 과정 암기]**
> > 단백질-DNA 결합을 분석하는 표준 기법입니다.
> >
> > **1. Workflow (P.29):**
> >     * **Cross-linking:** 포름알데히드로 단백질과 DNA를 딱 붙임(Fixation).
> >     * **Fragmentation:** DNA를 잘게 조각냄.
> >     * **Immunoprecipitation (IP):** 특정 단백질에 대한 **항체(Antibody)**로 낚아챔.
> >     * **DNA Extraction & Analysis:** 단백질을 녹여 없애고 DNA만 남겨서 PCR이나 Sequencing으로 분석.
> >     * *결과값: Enrichment = X/R (Input 대비 얼마나 농축됐는가)*
> >
> > **2. 주요 항체 리스트 (P.30 표):** *이거 구분 못하면 해석 못 합니다.*
> >     * **Active Markers (전사 활성):** H3K4me3 (Trimethyl), H3K9ac, H3K14ac (Acetylation은 주로 활성).
> >     * **Repressive Markers (전사 억제):** **H3K9me3**, **H3K27me3**. (꽉 뭉쳐서 유전자 꺼짐)
>
> > [!INFO] PAGE 31-32: 3C (Chromosome Conformation Capture)
> > 멀리 떨어진 DNA끼리의 상호작용(Looping 등)을 봅니다.
> > * **핵심 과정:** Cross-link -> **Restriction Enzyme (제한효소) 절단** -> **Ligation (연결)**.
> > * *포인트: 원래 멀리 있던 DNA라도, 단백질에 의해 묶여서 가까이 있다면 Ligation 되어 하나의 가닥으로 연결됩니다.*
> > * 결과물: Ligation products를 PCR/Sequencing하여 "아, 1번 염색체의 Promoter가 멀리 있는 Enhancer랑 붙어먹었구나"를 알아냅니다.

>
> > [!IMPORTANT] PAGE 36-37: 작동 기전 (Mechanism) **[핵심 원리]**
> > 박테리아의 적응 면역 시스템에서 유래했습니다.
> >
> > **1. 구성 요소:**
> >     * **Cas9 Nuclease:** 가위(Enzyme).
> >     * **sgRNA (single guide RNA):** **crRNA** (Target 인식 20nt) + **tracrRNA** (Cas9 결합 구조)를 하나로 합친 것.
> >     * **PAM (Protospacer Adjacent Motif):** **5'-NGG-3'**. Cas9이 DNA를 자르려면 타겟 서열 바로 뒤에 무조건 이 NGG가 있어야 합니다. (없으면 못 자름!)
> >
> > **2. 수선 기전 (Repair Pathways):** 잘린 DNA(DSB)는 두 가지로 수선됩니다.
> >     * **NHEJ (Non-homologous end joining):** 대충 갖다 붙임 -> 돌연변이(Indel) 발생 -> **Gene Knockout (유전자 파괴)**.
> >     * **HDR (Homology-directed repair):** 정상 DNA 조각(Donor DNA)을 넣어줌 -> 그걸 보고 베껴서 수선 -> **Gene Correction (유전자 교정/삽입)**.
>
> > [!IMPORTANT] PAGE 38-39: Cas9 구조와 변형 (dCas9) **[상세 구조]**
> > **1. Cas9의 도메인:**
> >     * **RuvC Domain:** 비표적 가닥(Non-target strand) 절단.
> >     * **HNH Domain:** 표적 가닥(Target strand) 절단.
> >     * 이 두 도메인이 모두 작동해야 이중 나선이 잘립니다(DSB).
> >
> > **2. dCas9 (dead Cas9): 가위 기능을 죽인 버전**
> >     * **변이(Mutation):** **D10A** (RuvC 불활성) + **H840A** (HNH 불활성).
> >     * **기능:** DNA를 자르지는 못하고, 가이드 RNA를 따라 **붙기만 합니다(Binding only)**.
> >     * **Nickase:** 둘 중 하나만 죽이면(예: D10A만), 한 가닥만 자르는 Nickase가 됩니다.
>
> > [!TIP] PAGE 40: dCas9의 응용 (CRISPRi & Epigenetics)
> > 자르지 못하는 dCas9에 다른 효소(Effector)를 달아서 씁니다.
> > * **CRISPRi (Interference):** dCas9이 프로모터에 딱 붙어서 전사 기구가 못 오게 막음 -> **Transcription OFF**.
> > * **Epigenetic Editing:**
> >     * **dCas9 + EZH2:** EZH2는 **H3K27 Methyltransferase**입니다. 타겟 유전자의 히스톤을 메틸화시켜 영구적으로 꺼버립니다(Silencing).
> >     * **dCas9 + GFP:** 특정 유전자 위치를 형광으로 추적(Live imaging).

> [!NOTE] Pages 42-51: 동물 실험 기초 및 형질전환 동물
> "왜 쥐를 쓰는가?"부터 "어떻게 유전자를 넣는가?"까지, 동물 실험의 A to Z입니다.
>
> > [!INFO] PAGE 42-43: 동물 실험 윤리 (3Rs)
> > 연구자의 기본 소양입니다. 무조건 외우세요.
> > * **Reduce:** 사용되는 동물의 **수(Number)**를 줄입니다.
> > * **Refine:** 동물에게 가해지는 **고통/스트레스(Stress)**를 최소화하도록 실험법을 개선합니다.
> > * **Replace:** 동물 실험을 세포 실험이나 시뮬레이션 등 **다른 방법(Other methods)**으로 대체합니다.
>
> > [!IMPORTANT] PAGE 44-45: 근교계 (Inbred Strains) **[필수 암기]**
> > 실험 편차를 줄이기 위해 유전자가 99% 이상 동일한 "순계" 마우스를 씁니다.
> > * **생성법:** **형매교배(Brother-sister mating)**를 **20세대** 이상 반복합니다.
> > * **주요 Strain 특성 (P.45 표):** *이거 족보입니다.*
> >     * **BALB/c:** 방사선 감수성(Sensitivity), 하이브리도마(Hybridoma) 제조에 사용.
> >     * **C57BL/6:** 방사선 조사 후 간암(Hepatoma) 발생 빈도 높음, 높은 보체(Complement) 활성.
> >     * **C3H/HeJ:** 유선 종양(Mammary tumor) 발생 빈도 높음.
> >     * **AKR/J:** 백혈병(Leukemia) 발생 빈도 높음.
> >     * **CBA/N:** X-linked 면역결핍(Immunodeficiency).
>
> > [!INFO] PAGE 46-47: Congenic Strains & Adoptive Transfer
> > * **Congenic (동계):** 딱 **하나의 유전자좌(Locus)**만 다르고 나머지 모든 유전자가 동일한 쥐입니다. 특정 유전자의 기능만 콕 집어 연구할 때 씁니다.
> > * **Adoptive Transfer (양자 전달):** 같은 Inbred strain끼리는 거부 반응이 없죠? 그래서 한 쥐의 림프구(Lymphocyte)를 분리해서 다른 쥐에게 주사하여 면역 반응을 전달하는 실험입니다.
>
> > [!IMPORTANT] PAGE 48-51: Transgenic Animals (형질전환) 생성법
> > 외부 유전자(Transgene)를 인위적으로 삽입하는 기술입니다.
> > * **핵심 과정 (Microinjection):**
> >     1.  수정란(Fertilized egg)의 **전핵(Pronucleus)**에 DNA를 직접 찔러 넣습니다.
> >     2.  이 수정란을 **가임신(Pseudopregnant)** 대리모의 난관에 이식합니다.
> >     3.  태어난 새끼의 **10~30%** 정도만 유전자를 가집니다.
> > * **한계점 (P.51):** 유전자가 게놈의 **무작위 위치(Randomly)**에 끼어들어갑니다. 엉뚱한 곳에 들어가서 암을 유발하거나(Tumor), 발현이 안 될 수도 있습니다.

> [!NOTE] Pages 52-55: 유전자 적중 (Gene Targeting / Knockout)
> Transgenic의 한계(무작위 삽입)를 극복하고, 원하는 유전자만 정확히 없애거나 교체하는 기술입니다. **"Homologous Recombination (상동 재조합)"**이 핵심입니다.
>
> > [!IMPORTANT] PAGE 52-54: 선별 과정 (Positive/Negative Selection) **[기전 상세]**
> > ES cell(배아줄기세포)에 유전자를 넣고, 제대로 들어간 놈만 어떻게 골라낼까요? **Neo**와 **TK** 유전자를 씁니다.
> >
> > 1.  **구성:** 타겟 유전자 중간에 **NeoR**(항생제 내성)을 넣고, 끝부분에 **tkHSV**(자살 유전자)를 달아둡니다.
> > 2.  **경우의 수 (P.54):**
> >     * **Case A (Homologous Recombination - 성공):** 타겟 위치에 정확히 교체됨. **NeoR은 들어가고, TK는 떨어져 나감.**
> >     * **Case B (Random Insertion - 실패):** 엉뚱한 곳에 통째로 들어감. **NeoR도 있고, TK도 있음.**
> >     * **Case C (No Insertion):** 아무것도 안 들어감.
> > 3.  **약물 처리 결과:**
> >     * **Neomycin 처리:** NeoR이 없는 Case C 사망.
> > 	* **Ganciclovir 처리:** TK가 있는 Case B 사망 (Ganciclovir가 TK에 의해 독성물질로 변함).
> >     * **최종 생존:** **Case A (성공한 세포)**만 살아남습니다.
>
> > [!INFO] PAGE 55: 키메라(Chimera) 마우스
> > * 유전자가 조작된 ES cell(검은 쥐 유래)을 흰 쥐의 배반포(Blastocyst)에 넣습니다.
> > * 태어난 쥐는 흰색/검은색이 섞인 **키메라**가 됩니다. 이 쥐를 다시 교배해서 온몸이 검은색인(생식세포까지 전달된) 쥐를 선별해야 완전한 Knockout 마우스가 됩니다.

> [!NOTE] Pages 56-63: Cre-Lox 시스템 (Inducible Deletion)
> 유전자를 그냥 없애버리면 태어나자마자 죽을 수도 있습니다(Lethal). 그래서 **"특정 시기, 특정 조직에서만"** 유전자를 없애는 고급 기술이 필요합니다.
>
> > [!IMPORTANT] PAGE 56-59: Cre-LoxP 원리 **[핵심 원리]**
> > * **도구:**
> >     * **Cre Recombinase:** P1 파지에서 유래한 효소. **loxP** 자리를 인식해 그 사이를 잘라냅니다.
> >     * **loxP:** 34bp의 특정 DNA 서열. 타겟 유전자 양옆에 붙여둡니다(Flanked).
> > * **전략 (Tissue-specific Knockout):**
> >     * 쥐 A: **DNA Polymerase $\beta$** 유전자 양옆에 **loxP**를 심어둠 (Floxed mouse).
> >     * 쥐 B: **Thymus-specific promoter** 뒤에 **Cre** 유전자를 달아둠.
> >     * 교배(A x B): 태어난 쥐는 온몸에 loxP가 있지만, 오직 **흉선(Thymus)**에서만 Cre가 나와서 해당 유전자를 잘라버립니다.
> >     * *결과: 다른 조직은 멀쩡하고, 흉선에서만 유전자가 결손됩니다.*
>
> > [!TIP] PAGE 60: Cre-Lox Turn-on (발현 유도)
> > 반대로 유전자를 켤 수도 있습니다.
> > * **전략:** **Promoter - [loxP - STOP - loxP] - Gene**
> > * 평소에는 STOP 코돈 때문에 발현이 안 됩니다.
> > * Cre가 작동하면 **STOP 코돈이 잘려 나가면서(Deletion)** 유전자가 켜집니다. 독성 유전자(Toxic gene) 연구에 씁니다.
>
> > [!IMPORTANT] PAGE 61-62: Cre-ER (Tamoxifen 유도) **[약물 기전]**
> > 조직 특이성에 **시간(Timing)** 조절까지 더한 기술입니다.
> > * **구조:** Cre 효소에 **Estrogen Receptor (ER)**를 붙여놨습니다.
> > * **평소 (Without Tamoxifen):** HSP90이라는 단백질이 Cre-ER를 잡고 있어서 핵 안으로 못 들어갑니다(Inactive).
> > * **약물 처리 (With Tamoxifen):** **Tamoxifen**을 주사하면 ER에 결합하고, HSP90이 떨어져 나갑니다.
> > * **결과:** 자유로워진 Cre-ER이 **핵(Nucleus)**으로 이동해 loxP를 자릅니다.
> > * *이로써 연구자가 원할 때(Time-specific) 유전자를 조작할 수 있게 됩니다.*

> [!abstract] 강의 요약: 면역학 연구기법 I (Flowchart)
>
> 이번 강의는 크게 **"시각화(Seeing) -> 분석(Analyzing) -> 조작(Manipulating)"**의 흐름으로 진행되었습니다.
>
> 1.  **Visualization (단백질과 세포 보기)**
>     * **고정된 조직:** 항체와 효소(HRP)를 이용한 **IHC**로 위치와 문맥 파악.
>     * **세밀한 구조:** 금 입자를 이용한 **IEM**으로 해상도 극대화.
>     * **살아있는 세포:** 형광(Fluorescence)과 **공초점/다광자 현미경**을 이용해 3D 및 실시간(Intravital) 관찰.
>
> 2.  **Genomic Analysis (유전자와 염색질 분석)**
>     * **위치 확인:** **FISH**를 통해 염색체 전좌(예: CML)나 유전자 거리를 시각화.
>     * **상호작용:** **ChIP**으로 단백질-DNA 결합을, **3C**로 DNA-DNA 상호작용을 분석.
>
> 3.  **Genetic Manipulation (유전자 조작 및 모델링)**
>     * **유전자 가위:** **CRISPR-Cas9**을 이용해 특정 유전자를 자르거나(KO), 조절(CRISPRi).
>     * **동물 모델:** **Transgenic** 및 **Knockout** 마우스 생성.
>     * **정밀 제어:** **Cre-Lox 시스템**을 이용해 "원하는 조직에서, 원하는 때에" 유전자를 조작.

> [!example] 핵심 키워드 정리 (Key Terms & Concepts)
>
> ### 1. Imaging & Microscopy
> * **IHC vs ICC:** 조직(Context) vs 세포(Localization).
> * **HRP & DAB:** 갈색(Brown) 발색의 표준. (cf. AP & pNPP는 Yellow).
> * **Stokes Shift:** 흡수 파장보다 방출 파장이 더 길다(에너지 손실).
> * **Fluorophores:** FITC(Green/525nm), PE(Red-Orange/Bright!), APC(Red/660nm).
> * **Confocal vs Multiphoton:**
>     * **Confocal:** **Pinhole**로 초점면 외 빛 차단 -> 선명한 상.
>     * **Multiphoton:** **장파장(적외선)** + 2개 광자 동시 타격 -> **Deep penetration** & **Low photobleaching**.
>
> ### 2. Chromatin & Genomics
> * **FISH:** 3D 구조 및 유전자 거리 측정 (CML의 **BCR-ABL** 전좌 확인).
> * **ChIP (Chromatin IP):** 단백질-DNA 결합 분석. (Enrichment = Input 대비 비율).
> * **Histone Markers:**
>     * Active: **H3K4me3**, Acetylation.
>     * Inactive: **H3K9me3**, **H3K27me3**.
>
> ### 3. CRISPR-Cas9
> * **Components:** Cas9(가위) + **sgRNA**(crRNA+tracrRNA) + **PAM**(5'-NGG-3').
> * **Repair:** **NHEJ**(Indel/KO) vs **HDR**(Correction).
> * **dCas9 (Dead Cas9):** **D10A & H840A** 변이. 자르지 않고 붙기만 함. (CRISPRi, Epigenetic editing 등에 응용).
>
> ### 4. Animal Models
> * **3Rs:** Replace, Reduce, Refine.
> * **Strains:**
>     * **Syngeneic (동계):** 유전자 동일 (Inbred).
>     * **Congenic:** 특정 유전자좌(Locus) 하나만 다름.
> * **Gene Targeting (Knockout):**
>     * **Homologous Recombination** 이용.
>     * **Selection:** **Positive(NeoR)** + **Negative(TK/Ganciclovir)**.
> * **Cre-Lox System (족보):**
>     * **Cre:** 자르는 효소 (Tissue-specific promoter).
>     * **loxP:** 잘리는 위치 (Target gene flanking).
>     * **Inducible:** **Cre-ER + Tamoxifen** (시간 조절 가능).

> [!NOTE] Pages 1-9: 항체 매개 시각화 기초 (IHC/ICC) 및 발색 원리
> 면역학 연구기법의 첫 단추는 **"보이지 않는 단백질을 항체로 염색해서 눈으로 보는 것"**입니다. 가장 기초적이면서도 임상/연구 모두에서 필수적인 파트입니다.
>
> > [!INFO] PAGE 1-2: 강의 목표 및 개요
> > * **표지 이미지(P.1):** 공초점 현미경 등으로 촬영된 세포들의 화려한 이미지가 보입니다. 앞으로 배울 기술들이죠.
> > * **핵심 목표(Goals, P.2):** 단순히 실험 방법만 외우는 게 아닙니다.
> >     1.  면역학 관련 문헌을 읽고 **방법론의 원리**를 이해해야 합니다.
> >     2.  연구자가 선택한 실험 기법이 **"적절한지"** 판단할 수 있는 비판적 사고가 필요합니다.
> >     3.  나만의 면역학적 의문을 해결할 **실험 설계(Experimental Design)** 능력을 길러야 합니다.
>
> > [!INFO] PAGE 3-4: 항체 매개 시각화 (개요)
> > * **주제:** 항체 매개 미세구조 시각화 (Antibody-Mediated Microscopic Visualization).
> > * **두 가지 큰 줄기 (P.4):**
> >     1.  **IHC/ICC:** 효소가 달린 항체(Enzyme-Conjugated Antibodies)를 이용해 **고정된 조직(Fixed Tissues)**의 이미지를 만듭니다.
> >     2.  **Immunoelectron Microscopy (면역전자현미경):** 금 입자(Gold Beads)를 사용하여 항체가 결합한 항원을 전자현미경으로 봅니다.
>
> > [!IMPORTANT] PAGE 5: ICC vs IHC (정의 및 차이)
> > 이 두 용어의 정의를 정확히 구분하세요. 대상(Target)이 다릅니다.
> > * **Immunocytochemistry (ICC, 면역세포화학법):**
> >     * 대상: **Cells** (배양 세포, 현탁액 세포).
> >     * 목적: 세포 내 특정 단백질(항원)의 **위치(Localization)**를 시각화.
> > * **Immunohistochemistry (IHC, 면역조직화학법):**
> >     * 대상: **Tissue section** (조직 절편).
> >     * 목적: 생체 조직(Biological tissues) 내에서 세포들의 분포와 항원의 위치를 **공간적 문맥(Spatial Context)** 속에서 파악.
> >     * *피라미드 그림을 보세요. Cells -> Tissues -> Organs 순으로 올라갑니다. Tissues 레벨을 보는 게 IHC입니다.*
>
> > [!IMPORTANT] PAGE 6: 효소와 기질 (Enzymes & Substrates) **[상세 암기]**
> > 항체 끝에 달린 **효소(Enzyme)**가 **기질(Substrate)**을 만나 **색깔(Color)**을 내는 화학 반응입니다. 표의 내용을 달달 외우세요.
> >
> > **1. HRP (Horseradish Peroxidase) 효소 기질:**
> > * **DAB (3,3'-diaminobenzidine):** **Brown (갈색)**. 가장 대표적입니다.
> > * **TMB (3,3',5,5'-tetramethylbenzidine):** **Blue** (흡광도 650nm). 주로 ELISA에 씁니다.
> > * **ABTS:** **Blue-green** (흡광도 410nm).
> > * **Luminol:** 발광(Light) 반응.
> >
> > **2. AP (Alkaline Phosphatase) 효소 기질:**
> > * **pNPP (p-Nitrophenyl Phosphate):** **Yellow** (흡광도 405nm).
> >
> > **3. Background (위양성 주의):**
> > * **적혈구(RBC):** Peroxidase가 많아서 HRP 사용 시 배경이 지저분해집니다.
> > * **폐포 세포(Alveolar cells):** AP가 많습니다.
> > * **유색 조직(Pigmented tissue):** 흑색종(Melanoma)처럼 원래 까만 조직은 갈색인 DAB를 쓰면 안 보입니다. 이때는 Purple, Red, Blue 계열 기질을 써야 합니다.
>
> > [!INFO] PAGE 7-8: IHC 데이터 해석 및 워크플로우
> > * **데이터 해석(P.7):**
> >     * **그림:** 제1형 당뇨병 모델(NOD mice). 파란색은 Insulin(베타세포), 갈색(Brown-red)은 CD8 T세포입니다.
> >     * **의미:** CD8 T세포가 인슐린을 만드는 베타세포를 공격(Killing)하여 파괴하는 자가면역 기전을 보여줍니다. (PD-L1과 PD-1 상호작용 관련 내용 포함)
> > * **실험 과정(P.8):**
> >     * Fixation(심장 관류) -> Dehydration -> Embedding(Paraffin/OCT) -> Sectioning -> Staining -> Mounting -> Imaging. 이 순서, 머릿속에 그리세요.
>
> > [!TIP] PAGE 9: 임상 판독 기준 (Scoring System) **[기출 포인트]**
> > 병리과에서 암 진단 시 사용하는 **HER2 IHC 판독 기준**입니다. 수치 싸움입니다.
> > * **Score 0 (Negative):** 염색 없음 혹은 **10% 미만**의 종양세포에서 불완전한 막 염색.
> > * **Score 1+ (Negative):** **10% 이상**에서 희미하게 보이나, **불완전한(Incomplete)** 막 염색.
> > * **Score 2+ (Weakly Positive):** **10% 이상**에서 약~중간 강도의 **완전한(Complete)** 막 염색.
> > * **Score 3+ (Positive):** **10% 이상**에서 **강한(Strong)** 완전한 막 염색.
> > * *핵심은 '10% 이상인가?' 그리고 '막이 완전히(Complete) 둘러싸였는가?'입니다.*

> [!NOTE] Pages 10-16: 면역전자현미경(IEM) & 형광 시각화
> 해상도(Resolution)의 끝판왕인 전자현미경과, 특정 분자를 빛나게 하는 형광 기법입니다.
>
> > [!INFO] PAGE 10: Immunoelectron Microscopy (IEM)
> > * **원리:** 항체에 색소 대신 **금 입자(Colloidal Gold Particle)**를 붙입니다. 금은 전자밀도(Electron-dense)가 높아 전자현미경에서 **까만 점(Black dots)**으로 보입니다.
> > * **다중 표지(Double Labeling):** 금 입자의 **크기(Size)**를 다르게 하여 두 가지 항원을 동시에 봅니다.
> >     * 예시 데이터: **MHC class II는 30nm**, **Class I은 15nm** 금 입자로 표지하여 세포 내 위치 관계를 분석했습니다.
>
> > [!INFO] PAGE 11: 면역형광(IF) 개요
> > * 단순히 보는 것을 넘어, 세포 내 분자(DNA 등)의 시각화, 생체 내(Intravital) 관찰까지 확장됩니다.
> > * 그림의 "Black Light Run"처럼 어둠 속에서 특정 부분만 빛나게 하는 원리입니다.
>
> > [!IMPORTANT] PAGE 12-13: 형광 원리 및 Fluorophores 스펙트럼 **[필수 암기]**
> > **1. Jablonski Diagram (P.12):**
> > * 빛을 흡수(Absorbance) -> 들뜬 상태(Excited) -> 에너지 일부 손실 -> 빛 방출(Emission).
> > * **Stokes Shift:** 방출되는 빛(Emission)은 흡수된 빛보다 항상 **파장이 깁니다(Longer wavelength)**. 에너지가 낮아졌으니까요.
> >
> > **2. 주요 형광물질 표 (P.13):** 짝을 지어 외우세요.
> > * **DAPI/Alexa 405:** Abs ~400nm / Em ~420nm (Blue). 핵 염색에 씀.
> > * **FITC (Fluorescein):** Abs **490nm** / Em **525nm** (Green). 가장 기본.
> > * **PE (Phycoerythrin):** Abs 490, 565 / Em **578nm** (Red-Orange). **밝기(Relative Brightness)가 5**로 매우 밝습니다.
> > * **APC (Allophycocyanin):** Abs **650nm** / Em **661nm** (Red).
> > * **Cy5:** Abs 650 / Em 670.
> > * *실험 설계 시 레이저 파장과 형광물질의 Absorbance가 맞는지 반드시 확인해야 합니다.*
>
> > [!INFO] PAGE 14: 형광현미경 필터 구조 (Filter Sets)
> > 원하는 파장만 골라내는 **3단계 필터링**이 핵심입니다.
> > 1.  **First barrier filter:** 광원에서 온 빛 중 특정 파장(예: Blue 450-490nm)만 통과시켜 시료를 때립니다(Excitation).
> > 2.  **Beam-splitting mirror:** 510nm 이하는 반사(Reflects), 그 이상은 통과(Transmits). 파장에 따라 길을 나눕니다.
> > 3.  **Second barrier filter:** 시료에서 나온 형광(예: Green 520-560nm)만 통과시키고, 나머지 잡광은 차단(Cuts out)하여 눈(Eyepiece)으로 보냅니다.
>
> > [!TIP] PAGE 15-16: GFP & RFP 그리고 Transgenic Animals
> > * **역사:** 1962년 Shimomura가 해파리(*Aequorea victoria*)에서 **GFP(Green)**를, 1999년 Lukyanov가 산호(Coral)에서 **RFP(Red)**를 발견했습니다. (2008년 노벨 화학상)
> > * **응용(Whole Animal):** 16페이지 사진을 보세요. **Actin promoter** 뒤에 GFP 유전자를 붙였습니다. Actin은 모든 세포에 있으니, 갓 태어난 쥐(Neonatal mice) 전체가 초록색으로 빛납니다.

> [!NOTE] Pages 17-21: 고급 이미징 (Confocal, Multiphoton, Intravital)
> 이제 **3차원(3D)**과 **살아있는(Live)** 상태를 보는 첨단 기법입니다.
>
> > [!IMPORTANT] PAGE 17: 공초점 현미경 (Confocal Microscopy)
> > * **핵심 부품:** **Pinhole (핀홀)**.
> > * **기전:** 초점면(Focal plane)에서 출발한 빛만 통과시키고, 초점면 위아래(Out of focus)에서 산란된 빛은 핀홀 주변에서 **차단**됩니다.
> > * **결과:** 흐릿함이 없는 **선명한 상**을 얻고, Z축 스캔을 통해 **3차원 영상** 제작이 가능합니다.
>
> > [!IMPORTANT] PAGE 18: 다광자 현미경 (Multiphoton Microscopy)
> > 공초점의 업그레이드 버전입니다.
> > * **원리:** **장파장/저에너지(Long wavelength, Low energy)**의 적외선 레이저를 씁니다.
> > * **Two-photon Excitation:** 광자 1개로는 에너지가 부족해서, 광자 2개가 동시에 때려야만 형광이 나옵니다. 즉, 초점이 맺힌 점(Focus spot) 외에는 아예 형광이 안 생깁니다. (핀홀 불필요)
> > * **장점:**
> >     1.  **광퇴색(Photobleaching) 최소화:** 에너지가 낮아 검체 손상이 적습니다.
> >     2.  **깊은 투과력:** 장파장은 조직 깊숙이 들어갈 수 있습니다.
>
> > [!TIP] PAGE 19-21: 생체 내 영상 (Intravital Imaging) **[데이터 해석]**
> > 살아있는 동물에서 면역세포의 움직임을 실시간으로 봅니다.
> >
> > **1. 혈관 투과성 실험 (P.19):**
> > * **설계:** 귀 진피(Ear dermis)에 **VEGFA164**(혈관내피성장인자) 주입. 혈관 내에는 **2000 kDa FITC-Dextran**(녹색) 주입.
> > * **결과:** VEGFA 주입 후 혈관 직경이 커지고, 녹색 형광이 혈관 밖으로 새어 나가는 **Vascular leakage**가 관찰됨.
> >
> > **2. 호중구의 박테리아 제거 (P.20):**
> > * **상황:** UPEC(대장균) 감염 모델.
> > * **관찰:** 0분부터 **150분**까지 촬영. 호중구(Neutrophil)가 몰려와 박테리아(형광)를 포식하여 제거(Clearance)하는 과정을 시각화.
> >
> > **3. 림프절 T세포 이동 (P.21):**
> > * **표지:** T세포는 **CFSE (Green)**로, 고내피세동맥(HEV)은 **Red**로 염색.
> > * **결과:** T세포가 HEV를 통해 림프절로 들어와 이동하는 3차원 궤적(Trafficking)을 확인.

> 유전자의 위치를 눈으로 확인하고, 유전자가 감겨 있는 히스톤의 상태를 분석하는 파트입니다.
>
> > [!INFO] PAGE 22-23: 3-D FISH (Fluorescence in situ hybridization)
> > * **정의:** 형광 RNA 탐침(Probe)을 DNA에 붙여서 염색체의 3차원적 구조와 거리를 측정합니다.
> > * **거리 측정 데이터 (P.23):**
> >     * **T cells:** $V_H$ (Red)와 $V_{H}J558$ (Green) 유전자 거리가 멉니다(Ready discriminated). 아직 재조합 전이라 그렇습니다.
> >     * **Pro-B cells:** 면역글로불린 유전자좌(Ig locus)가 **수축(Contraction)**합니다. 두 유전자(Red/Green)가 가까워져서 색이 겹쳐 보일 정도입니다. 이게 유전자 재조합의 전조 현상입니다.
>
> > [!IMPORTANT] PAGE 24: 만성 골수성 백혈병 (CML)과 필라델피아 염색체 **[병리 연계]**
> > 이 그림은 족보 중의 족보입니다.
> > * **정상:** 9번(Green)과 22번(Red) 염색체는 떨어져 있어야 합니다.
> > * **CML (Chronic Myeloid Leukemia):**
> >     * 9번과 22번 사이에 **상호 전좌(Reciprocal Translocation)**가 일어납니다.
> >     * **결과:** 9번의 **ABL** 유전자와 22번의 **BCR** 유전자가 합쳐져 **BCR-ABL** 융합 유전자가 생깁니다.
> >     * **병리:** 이 융합 단백질은 **Tyrosine Kinase 활성**이 비정상적으로 높아져 암(Tumor)을 유발합니다. (이걸 표적으로 한 약이 글리벡입니다.)
> >     * *그림 B/C를 보면 Red/Green 점이 붙어서 노란색/중간색으로 보이는 게 전좌의 증거입니다.*
>
> > [!INFO] PAGE 26-27: 염색질 구조 (Chromatin Structure)
> > * **구조:** DNA는 히스톤(Histone)을 감아 **뉴클레오솜(Nucleosome)**을 형성합니다. (Beads on a string)
> > * **히스톤 꼬리 변형 (Histone Tail Modification):**
> >     * **Acetylation (아세틸화):** 히스톤과 DNA 결합을 느슨하게 함 -> **Open Chromatin** -> **Gene Active**. (전사 기구가 접근 가능)
> >     * **Methylation (메틸화):** 주로 **Closed Chromatin** -> **Gene Inactive**. (단, 위치에 따라 다름)

> [!NOTE] Pages 28-32: 단백질-DNA 상호작용 분석 (ChIP & 3C)
> "이 단백질이 DNA 어디에 붙었니?"(ChIP)와 "이 DNA가 저 DNA랑 만났니?"(3C)를 묻는 기술입니다.
>
> > [!IMPORTANT] PAGE 28-30: ChIP (Chromatin Immunoprecipitation) **[실험 과정 암기]**
> > 단백질-DNA 결합을 분석하는 표준 기법입니다.
> >
> > **1. Workflow (P.29):**
> >     * **Cross-linking:** 포름알데히드로 단백질과 DNA를 딱 붙임(Fixation).
> >     * **Fragmentation:** DNA를 잘게 조각냄.
> >     * **Immunoprecipitation (IP):** 특정 단백질에 대한 **항체(Antibody)**로 낚아챔.
> >     * **DNA Extraction & Analysis:** 단백질을 녹여 없애고 DNA만 남겨서 PCR이나 Sequencing으로 분석.
> >     * *결과값: Enrichment = X/R (Input 대비 얼마나 농축됐는가)*
> >
> > **2. 주요 항체 리스트 (P.30 표):** *이거 구분 못하면 해석 못 합니다.*
> >     * **Active Markers (전사 활성):** H3K4me3 (Trimethyl), H3K9ac, H3K14ac (Acetylation은 주로 활성).
> >     * **Repressive Markers (전사 억제):** **H3K9me3**, **H3K27me3**. (꽉 뭉쳐서 유전자 꺼짐)
>
> > [!INFO] PAGE 31-32: 3C (Chromosome Conformation Capture)
> > 멀리 떨어진 DNA끼리의 상호작용(Looping 등)을 봅니다.
> > * **핵심 과정:** Cross-link -> **Restriction Enzyme (제한효소) 절단** -> **Ligation (연결)**.
> > * *포인트: 원래 멀리 있던 DNA라도, 단백질에 의해 묶여서 가까이 있다면 Ligation 되어 하나의 가닥으로 연결됩니다.*
> > * 결과물: Ligation products를 PCR/Sequencing하여 "아, 1번 염색체의 Promoter가 멀리 있는 Enhancer랑 붙어먹었구나"를 알아냅니다.

> [!NOTE] Pages 33-41: 유전자 가위 (CRISPR-Cas9)
> 2020년 노벨 화학상의 주인공입니다. 단순한 가위질을 넘어 유전자 조절까지 확장된 최신 경향을 파악하세요.
>
> > [!INFO] PAGE 33-35: 역사 (Timeline)
> > * **1987:** 대장균(E. coli)에서 CRISPR 서열(Clustered repeats) 최초 발견 (이땐 뭔지 몰랐음).
> > * **2012:** **Doudna & Charpentier** (노벨상 주역)가 Cas9의 DNA 절단 기전 규명.
> > * **2013:** 인간 세포(Mammalian cells)에서 게놈 교정 성공.
> > * **2020:** 노벨 화학상 수상.
>
> > [!IMPORTANT] PAGE 36-37: 작동 기전 (Mechanism) **[핵심 원리]**
> > 박테리아의 적응 면역 시스템에서 유래했습니다.
> >
> > **1. 구성 요소:**
> >     * **Cas9 Nuclease:** 가위(Enzyme).
> >     * **sgRNA (single guide RNA):** **crRNA** (Target 인식 20nt) + **tracrRNA** (Cas9 결합 구조)를 하나로 합친 것.
> >     * **PAM (Protospacer Adjacent Motif):** **5'-NGG-3'**. Cas9이 DNA를 자르려면 타겟 서열 바로 뒤에 무조건 이 NGG가 있어야 합니다. (없으면 못 자름!)
> >
> > **2. 수선 기전 (Repair Pathways):** 잘린 DNA(DSB)는 두 가지로 수선됩니다.
> >     * **NHEJ (Non-homologous end joining):** 대충 갖다 붙임 -> 돌연변이(Indel) 발생 -> **Gene Knockout (유전자 파괴)**.
> >     * **HDR (Homology-directed repair):** 정상 DNA 조각(Donor DNA)을 넣어줌 -> 그걸 보고 베껴서 수선 -> **Gene Correction (유전자 교정/삽입)**.
>
> > [!IMPORTANT] PAGE 38-39: Cas9 구조와 변형 (dCas9) **[상세 구조]**
> > **1. Cas9의 도메인:**
> >     * **RuvC Domain:** 비표적 가닥(Non-target strand) 절단.
> >     * **HNH Domain:** 표적 가닥(Target strand) 절단.
> >     * 이 두 도메인이 모두 작동해야 이중 나선이 잘립니다(DSB).
> >
> > **2. dCas9 (dead Cas9): 가위 기능을 죽인 버전**
> >     * **변이(Mutation):** **D10A** (RuvC 불활성) + **H840A** (HNH 불활성).
> >     * **기능:** DNA를 자르지는 못하고, 가이드 RNA를 따라 **붙기만 합니다(Binding only)**.
> >     * **Nickase:** 둘 중 하나만 죽이면(예: D10A만), 한 가닥만 자르는 Nickase가 됩니다.
>
> > [!TIP] PAGE 40: dCas9의 응용 (CRISPRi & Epigenetics)
> > 자르지 못하는 dCas9에 다른 효소(Effector)를 달아서 씁니다.
> > * **CRISPRi (Interference):** dCas9이 프로모터에 딱 붙어서 전사 기구가 못 오게 막음 -> **Transcription OFF**.
> > * **Epigenetic Editing:**
> >     * **dCas9 + EZH2:** EZH2는 **H3K27 Methyltransferase**입니다. 타겟 유전자의 히스톤을 메틸화시켜 영구적으로 꺼버립니다(Silencing).
> >     * **dCas9 + GFP:** 특정 유전자 위치를 형광으로 추적(Live imaging).

>
> > [!INFO] PAGE 42-43: 동물 실험 윤리 (3Rs)
> > 연구자의 기본 소양입니다. 무조건 외우세요.
> > * **Reduce:** 사용되는 동물의 **수(Number)**를 줄입니다.
> > * **Refine:** 동물에게 가해지는 **고통/스트레스(Stress)**를 최소화하도록 실험법을 개선합니다.
> > * **Replace:** 동물 실험을 세포 실험이나 시뮬레이션 등 **다른 방법(Other methods)**으로 대체합니다.
>
> > [!IMPORTANT] PAGE 44-45: 근교계 (Inbred Strains) **[필수 암기]**
> > 실험 편차를 줄이기 위해 유전자가 99% 이상 동일한 "순계" 마우스를 씁니다.
> > * **생성법:** **형매교배(Brother-sister mating)**를 **20세대** 이상 반복합니다.
> > * **주요 Strain 특성 (P.45 표):** *이거 족보입니다.*
> >     * **BALB/c:** 방사선 감수성(Sensitivity), 하이브리도마(Hybridoma) 제조에 사용.
> >     * **C57BL/6:** 방사선 조사 후 간암(Hepatoma) 발생 빈도 높음, 높은 보체(Complement) 활성.
> >     * **C3H/HeJ:** 유선 종양(Mammary tumor) 발생 빈도 높음.
> >     * **AKR/J:** 백혈병(Leukemia) 발생 빈도 높음.
> >     * **CBA/N:** X-linked 면역결핍(Immunodeficiency).
>
> > [!INFO] PAGE 46-47: Congenic Strains & Adoptive Transfer
> > * **Congenic (동계):** 딱 **하나의 유전자좌(Locus)**만 다르고 나머지 모든 유전자가 동일한 쥐입니다. 특정 유전자의 기능만 콕 집어 연구할 때 씁니다.
> > * **Adoptive Transfer (양자 전달):** 같은 Inbred strain끼리는 거부 반응이 없죠? 그래서 한 쥐의 림프구(Lymphocyte)를 분리해서 다른 쥐에게 주사하여 면역 반응을 전달하는 실험입니다.
>
> > [!IMPORTANT] PAGE 48-51: Transgenic Animals (형질전환) 생성법
> > 외부 유전자(Transgene)를 인위적으로 삽입하는 기술입니다.
> > * **핵심 과정 (Microinjection):**
> >     1.  수정란(Fertilized egg)의 **전핵(Pronucleus)**에 DNA를 직접 찔러 넣습니다.
> >     2.  이 수정란을 **가임신(Pseudopregnant)** 대리모의 난관에 이식합니다.
> >     3.  태어난 새끼의 **10~30%** 정도만 유전자를 가집니다.
> > * **한계점 (P.51):** 유전자가 게놈의 **무작위 위치(Randomly)**에 끼어들어갑니다. 엉뚱한 곳에 들어가서 암을 유발하거나(Tumor), 발현이 안 될 수도 있습니다.

> [!NOTE] Pages 52-55: 유전자 적중 (Gene Targeting / Knockout)
> Transgenic의 한계(무작위 삽입)를 극복하고, 원하는 유전자만 정확히 없애거나 교체하는 기술입니다. **"Homologous Recombination (상동 재조합)"**이 핵심입니다.
>
> > [!IMPORTANT] PAGE 52-54: 선별 과정 (Positive/Negative Selection) **[기전 상세]**
> > ES cell(배아줄기세포)에 유전자를 넣고, 제대로 들어간 놈만 어떻게 골라낼까요? **Neo**와 **TK** 유전자를 씁니다.
> >
> > 1.  **구성:** 타겟 유전자 중간에 **NeoR**(항생제 내성)을 넣고, 끝부분에 **tkHSV**(자살 유전자)를 달아둡니다.
> > 2.  **경우의 수 (P.54):**
> >     * **Case A (Homologous Recombination - 성공):** 타겟 위치에 정확히 교체됨. **NeoR은 들어가고, TK는 떨어져 나감.**
> >     * **Case B (Random Insertion - 실패):** 엉뚱한 곳에 통째로 들어감. **NeoR도 있고, TK도 있음.**
> >     * **Case C (No Insertion):** 아무것도 안 들어감.
> > 3.  **약물 처리 결과:**
> >     * **Neomycin 처리:** NeoR이 없는 Case C 사망.
> > 	* **Ganciclovir 처리:** TK가 있는 Case B 사망 (Ganciclovir가 TK에 의해 독성물질로 변함).
> >     * **최종 생존:** **Case A (성공한 세포)**만 살아남습니다.
>
> > [!INFO] PAGE 55: 키메라(Chimera) 마우스
> > * 유전자가 조작된 ES cell(검은 쥐 유래)을 흰 쥐의 배반포(Blastocyst)에 넣습니다.
> > * 태어난 쥐는 흰색/검은색이 섞인 **키메라**가 됩니다. 이 쥐를 다시 교배해서 온몸이 검은색인(생식세포까지 전달된) 쥐를 선별해야 완전한 Knockout 마우스가 됩니다.

> [!NOTE] Pages 56-63: Cre-Lox 시스템 (Inducible Deletion)
> 유전자를 그냥 없애버리면 태어나자마자 죽을 수도 있습니다(Lethal). 그래서 **"특정 시기, 특정 조직에서만"** 유전자를 없애는 고급 기술이 필요합니다.
>
> > [!IMPORTANT] PAGE 56-59: Cre-LoxP 원리 **[핵심 원리]**
> > * **도구:**
> >     * **Cre Recombinase:** P1 파지에서 유래한 효소. **loxP** 자리를 인식해 그 사이를 잘라냅니다.
> >     * **loxP:** 34bp의 특정 DNA 서열. 타겟 유전자 양옆에 붙여둡니다(Flanked).
> > * **전략 (Tissue-specific Knockout):**
> >     * 쥐 A: **DNA Polymerase $\beta$** 유전자 양옆에 **loxP**를 심어둠 (Floxed mouse).
> >     * 쥐 B: **Thymus-specific promoter** 뒤에 **Cre** 유전자를 달아둠.
> >     * 교배(A x B): 태어난 쥐는 온몸에 loxP가 있지만, 오직 **흉선(Thymus)**에서만 Cre가 나와서 해당 유전자를 잘라버립니다.
> >     * *결과: 다른 조직은 멀쩡하고, 흉선에서만 유전자가 결손됩니다.*
>
> > [!TIP] PAGE 60: Cre-Lox Turn-on (발현 유도)
> > 반대로 유전자를 켤 수도 있습니다.
> > * **전략:** **Promoter - [loxP - STOP - loxP] - Gene**
> > * 평소에는 STOP 코돈 때문에 발현이 안 됩니다.
> > * Cre가 작동하면 **STOP 코돈이 잘려 나가면서(Deletion)** 유전자가 켜집니다. 독성 유전자(Toxic gene) 연구에 씁니다.
>
> > [!IMPORTANT] PAGE 61-62: Cre-ER (Tamoxifen 유도) **[약물 기전]**
> > 조직 특이성에 **시간(Timing)** 조절까지 더한 기술입니다.
> > * **구조:** Cre 효소에 **Estrogen Receptor (ER)**를 붙여놨습니다.
> > * **평소 (Without Tamoxifen):** HSP90이라는 단백질이 Cre-ER를 잡고 있어서 핵 안으로 못 들어갑니다(Inactive).
> > * **약물 처리 (With Tamoxifen):** **Tamoxifen**을 주사하면 ER에 결합하고, HSP90이 떨어져 나갑니다.
> > * **결과:** 자유로워진 Cre-ER이 **핵(Nucleus)**으로 이동해 loxP를 자릅니다.
> > * *이로써 연구자가 원할 때(Time-specific) 유전자를 조작할 수 있게 됩니다.*

> [!abstract] 강의 요약: 면역학 연구기법 I (Flowchart)
>
> 이번 강의는 크게 **"시각화(Seeing) -> 분석(Analyzing) -> 조작(Manipulating)"**의 흐름으로 진행되었습니다.
>
> 1.  **Visualization (단백질과 세포 보기)**
>     * **고정된 조직:** 항체와 효소(HRP)를 이용한 **IHC**로 위치와 문맥 파악.
>     * **세밀한 구조:** 금 입자를 이용한 **IEM**으로 해상도 극대화.
>     * **살아있는 세포:** 형광(Fluorescence)과 **공초점/다광자 현미경**을 이용해 3D 및 실시간(Intravital) 관찰.
>
> 2.  **Genomic Analysis (유전자와 염색질 분석)**
>     * **위치 확인:** **FISH**를 통해 염색체 전좌(예: CML)나 유전자 거리를 시각화.
>     * **상호작용:** **ChIP**으로 단백질-DNA 결합을, **3C**로 DNA-DNA 상호작용을 분석.
>
> 3.  **Genetic Manipulation (유전자 조작 및 모델링)**
>     * **유전자 가위:** **CRISPR-Cas9**을 이용해 특정 유전자를 자르거나(KO), 조절(CRISPRi).
>     * **동물 모델:** **Transgenic** 및 **Knockout** 마우스 생성.
>     * **정밀 제어:** **Cre-Lox 시스템**을 이용해 "원하는 조직에서, 원하는 때에" 유전자를 조작.

> [!example] 핵심 키워드 정리 (Key Terms & Concepts)
>
> ### 1. Imaging & Microscopy
> * **IHC vs ICC:** 조직(Context) vs 세포(Localization).
> * **HRP & DAB:** 갈색(Brown) 발색의 표준. (cf. AP & pNPP는 Yellow).
> * **Stokes Shift:** 흡수 파장보다 방출 파장이 더 길다(에너지 손실).
> * **Fluorophores:** FITC(Green/525nm), PE(Red-Orange/Bright!), APC(Red/660nm).
> * **Confocal vs Multiphoton:**
>     * **Confocal:** **Pinhole**로 초점면 외 빛 차단 -> 선명한 상.
>     * **Multiphoton:** **장파장(적외선)** + 2개 광자 동시 타격 -> **Deep penetration** & **Low photobleaching**.
>
> ### 2. Chromatin & Genomics
> * **FISH:** 3D 구조 및 유전자 거리 측정 (CML의 **BCR-ABL** 전좌 확인).
> * **ChIP (Chromatin IP):** 단백질-DNA 결합 분석. (Enrichment = Input 대비 비율).
> * **Histone Markers:**
>     * Active: **H3K4me3**, Acetylation.
>     * Inactive: **H3K9me3**, **H3K27me3**.
>
> ### 3. CRISPR-Cas9
> * **Components:** Cas9(가위) + **sgRNA**(crRNA+tracrRNA) + **PAM**(5'-NGG-3').
> * **Repair:** **NHEJ**(Indel/KO) vs **HDR**(Correction).
> * **dCas9 (Dead Cas9):** **D10A & H840A** 변이. 자르지 않고 붙기만 함. (CRISPRi, Epigenetic editing 등에 응용).
>
> ### 4. Animal Models
> * **3Rs:** Replace, Reduce, Refine.
> * **Strains:**
>     * **Syngeneic (동계):** 유전자 동일 (Inbred).
>     * **Congenic:** 특정 유전자좌(Locus) 하나만 다름.
> * **Gene Targeting (Knockout):**
>     * **Homologous Recombination** 이용.
>     * **Selection:** **Positive(NeoR)** + **Negative(TK/Ganciclovir)**.
> * **Cre-Lox System (족보):**
>     * **Cre:** 자르는 효소 (Tissue-specific promoter).
>     * **loxP:** 잘리는 위치 (Target gene flanking).
>     * **Inducible:** **Cre-ER + Tamoxifen** (시간 조절 가능).

> [!NOTE] Pages 1-9: 항체 매개 시각화 기초 (IHC/ICC) 및 발색 원리
> 면역학 연구기법의 첫 단추는 **"보이지 않는 단백질을 항체로 염색해서 눈으로 보는 것"**입니다. 가장 기초적이면서도 임상/연구 모두에서 필수적인 파트입니다.
>
> > [!INFO] PAGE 1-2: 강의 목표 및 개요
> > * **표지 이미지(P.1):** 공초점 현미경 등으로 촬영된 세포들의 화려한 이미지가 보입니다. 앞으로 배울 기술들이죠.
> > * **핵심 목표(Goals, P.2):** 단순히 실험 방법만 외우는 게 아닙니다.
> >     1.  면역학 관련 문헌을 읽고 **방법론의 원리**를 이해해야 합니다.
> >     2.  연구자가 선택한 실험 기법이 **"적절한지"** 판단할 수 있는 비판적 사고가 필요합니다.
> >     3.  나만의 면역학적 의문을 해결할 **실험 설계(Experimental Design)** 능력을 길러야 합니다.
>
> > [!INFO] PAGE 3-4: 항체 매개 시각화 (개요)
> > * **주제:** 항체 매개 미세구조 시각화 (Antibody-Mediated Microscopic Visualization).
> > * **두 가지 큰 줄기 (P.4):**
> >     1.  **IHC/ICC:** 효소가 달린 항체(Enzyme-Conjugated Antibodies)를 이용해 **고정된 조직(Fixed Tissues)**의 이미지를 만듭니다.
> >     2.  **Immunoelectron Microscopy (면역전자현미경):** 금 입자(Gold Beads)를 사용하여 항체가 결합한 항원을 전자현미경으로 봅니다.
>
> > [!IMPORTANT] PAGE 5: ICC vs IHC (정의 및 차이)
> > 이 두 용어의 정의를 정확히 구분하세요. 대상(Target)이 다릅니다.
> > * **Immunocytochemistry (ICC, 면역세포화학법):**
> >     * 대상: **Cells** (배양 세포, 현탁액 세포).
> >     * 목적: 세포 내 특정 단백질(항원)의 **위치(Localization)**를 시각화.
> > * **Immunohistochemistry (IHC, 면역조직화학법):**
> >     * 대상: **Tissue section** (조직 절편).
> >     * 목적: 생체 조직(Biological tissues) 내에서 세포들의 분포와 항원의 위치를 **공간적 문맥(Spatial Context)** 속에서 파악.
> >     * *피라미드 그림을 보세요. Cells -> Tissues -> Organs 순으로 올라갑니다. Tissues 레벨을 보는 게 IHC입니다.*
>
> > [!IMPORTANT] PAGE 6: 효소와 기질 (Enzymes & Substrates) **[상세 암기]**
> > 항체 끝에 달린 **효소(Enzyme)**가 **기질(Substrate)**을 만나 **색깔(Color)**을 내는 화학 반응입니다. 표의 내용을 달달 외우세요.
> >
> > **1. HRP (Horseradish Peroxidase) 효소 기질:**
> > * **DAB (3,3'-diaminobenzidine):** **Brown (갈색)**. 가장 대표적입니다.
> > * **TMB (3,3',5,5'-tetramethylbenzidine):** **Blue** (흡광도 650nm). 주로 ELISA에 씁니다.
> > * **ABTS:** **Blue-green** (흡광도 410nm).
> > * **Luminol:** 발광(Light) 반응.
> >
> > **2. AP (Alkaline Phosphatase) 효소 기질:**
> > * **pNPP (p-Nitrophenyl Phosphate):** **Yellow** (흡광도 405nm).
> >
> > **3. Background (위양성 주의):**
> > * **적혈구(RBC):** Peroxidase가 많아서 HRP 사용 시 배경이 지저분해집니다.
> > * **폐포 세포(Alveolar cells):** AP가 많습니다.
> > * **유색 조직(Pigmented tissue):** 흑색종(Melanoma)처럼 원래 까만 조직은 갈색인 DAB를 쓰면 안 보입니다. 이때는 Purple, Red, Blue 계열 기질을 써야 합니다.
>
> > [!INFO] PAGE 7-8: IHC 데이터 해석 및 워크플로우
> > * **데이터 해석(P.7):**
> >     * **그림:** 제1형 당뇨병 모델(NOD mice). 파란색은 Insulin(베타세포), 갈색(Brown-red)은 CD8 T세포입니다.
> >     * **의미:** CD8 T세포가 인슐린을 만드는 베타세포를 공격(Killing)하여 파괴하는 자가면역 기전을 보여줍니다. (PD-L1과 PD-1 상호작용 관련 내용 포함)
> > * **실험 과정(P.8):**
> >     * Fixation(심장 관류) -> Dehydration -> Embedding(Paraffin/OCT) -> Sectioning -> Staining -> Mounting -> Imaging. 이 순서, 머릿속에 그리세요.
>
> > [!TIP] PAGE 9: 임상 판독 기준 (Scoring System) **[기출 포인트]**
> > 병리과에서 암 진단 시 사용하는 **HER2 IHC 판독 기준**입니다. 수치 싸움입니다.
> > * **Score 0 (Negative):** 염색 없음 혹은 **10% 미만**의 종양세포에서 불완전한 막 염색.
> > * **Score 1+ (Negative):** **10% 이상**에서 희미하게 보이나, **불완전한(Incomplete)** 막 염색.
> > * **Score 2+ (Weakly Positive):** **10% 이상**에서 약~중간 강도의 **완전한(Complete)** 막 염색.
> > * **Score 3+ (Positive):** **10% 이상**에서 **강한(Strong)** 완전한 막 염색.
> > * *핵심은 '10% 이상인가?' 그리고 '막이 완전히(Complete) 둘러싸였는가?'입니다.*

> [!NOTE] Pages 10-16: 면역전자현미경(IEM) & 형광 시각화
> 해상도(Resolution)의 끝판왕인 전자현미경과, 특정 분자를 빛나게 하는 형광 기법입니다.
>
> > [!INFO] PAGE 10: Immunoelectron Microscopy (IEM)
> > * **원리:** 항체에 색소 대신 **금 입자(Colloidal Gold Particle)**를 붙입니다. 금은 전자밀도(Electron-dense)가 높아 전자현미경에서 **까만 점(Black dots)**으로 보입니다.
> > * **다중 표지(Double Labeling):** 금 입자의 **크기(Size)**를 다르게 하여 두 가지 항원을 동시에 봅니다.
> >     * 예시 데이터: **MHC class II는 30nm**, **Class I은 15nm** 금 입자로 표지하여 세포 내 위치 관계를 분석했습니다.
>
> > [!INFO] PAGE 11: 면역형광(IF) 개요
> > * 단순히 보는 것을 넘어, 세포 내 분자(DNA 등)의 시각화, 생체 내(Intravital) 관찰까지 확장됩니다.
> > * 그림의 "Black Light Run"처럼 어둠 속에서 특정 부분만 빛나게 하는 원리입니다.
>
> > [!IMPORTANT] PAGE 12-13: 형광 원리 및 Fluorophores 스펙트럼 **[필수 암기]**
> > **1. Jablonski Diagram (P.12):**
> > * 빛을 흡수(Absorbance) -> 들뜬 상태(Excited) -> 에너지 일부 손실 -> 빛 방출(Emission).
> > * **Stokes Shift:** 방출되는 빛(Emission)은 흡수된 빛보다 항상 **파장이 깁니다(Longer wavelength)**. 에너지가 낮아졌으니까요.
> >
> > **2. 주요 형광물질 표 (P.13):** 짝을 지어 외우세요.
> > * **DAPI/Alexa 405:** Abs ~400nm / Em ~420nm (Blue). 핵 염색에 씀.
> > * **FITC (Fluorescein):** Abs **490nm** / Em **525nm** (Green). 가장 기본.
> > * **PE (Phycoerythrin):** Abs 490, 565 / Em **578nm** (Red-Orange). **밝기(Relative Brightness)가 5**로 매우 밝습니다.
> > * **APC (Allophycocyanin):** Abs **650nm** / Em **661nm** (Red).
> > * **Cy5:** Abs 650 / Em 670.
> > * *실험 설계 시 레이저 파장과 형광물질의 Absorbance가 맞는지 반드시 확인해야 합니다.*
>
> > [!INFO] PAGE 14: 형광현미경 필터 구조 (Filter Sets)
> > 원하는 파장만 골라내는 **3단계 필터링**이 핵심입니다.
> > 1.  **First barrier filter:** 광원에서 온 빛 중 특정 파장(예: Blue 450-490nm)만 통과시켜 시료를 때립니다(Excitation).
> > 2.  **Beam-splitting mirror:** 510nm 이하는 반사(Reflects), 그 이상은 통과(Transmits). 파장에 따라 길을 나눕니다.
> > 3.  **Second barrier filter:** 시료에서 나온 형광(예: Green 520-560nm)만 통과시키고, 나머지 잡광은 차단(Cuts out)하여 눈(Eyepiece)으로 보냅니다.
>
> > [!TIP] PAGE 15-16: GFP & RFP 그리고 Transgenic Animals
> > * **역사:** 1962년 Shimomura가 해파리(*Aequorea victoria*)에서 **GFP(Green)**를, 1999년 Lukyanov가 산호(Coral)에서 **RFP(Red)**를 발견했습니다. (2008년 노벨 화학상)
> > * **응용(Whole Animal):** 16페이지 사진을 보세요. **Actin promoter** 뒤에 GFP 유전자를 붙였습니다. Actin은 모든 세포에 있으니, 갓 태어난 쥐(Neonatal mice) 전체가 초록색으로 빛납니다.

> [!NOTE] Pages 17-21: 고급 이미징 (Confocal, Multiphoton, Intravital)
> 이제 **3차원(3D)**과 **살아있는(Live)** 상태를 보는 첨단 기법입니다.
>
> > [!IMPORTANT] PAGE 17: 공초점 현미경 (Confocal Microscopy)
> > * **핵심 부품:** **Pinhole (핀홀)**.
> > * **기전:** 초점면(Focal plane)에서 출발한 빛만 통과시키고, 초점면 위아래(Out of focus)에서 산란된 빛은 핀홀 주변에서 **차단**됩니다.
> > * **결과:** 흐릿함이 없는 **선명한 상**을 얻고, Z축 스캔을 통해 **3차원 영상** 제작이 가능합니다.
>
> > [!IMPORTANT] PAGE 18: 다광자 현미경 (Multiphoton Microscopy)
> > 공초점의 업그레이드 버전입니다.
> > * **원리:** **장파장/저에너지(Long wavelength, Low energy)**의 적외선 레이저를 씁니다.
> > * **Two-photon Excitation:** 광자 1개로는 에너지가 부족해서, 광자 2개가 동시에 때려야만 형광이 나옵니다. 즉, 초점이 맺힌 점(Focus spot) 외에는 아예 형광이 안 생깁니다. (핀홀 불필요)
> > * **장점:**
> >     1.  **광퇴색(Photobleaching) 최소화:** 에너지가 낮아 검체 손상이 적습니다.
> >     2.  **깊은 투과력:** 장파장은 조직 깊숙이 들어갈 수 있습니다.
>
> > [!TIP] PAGE 19-21: 생체 내 영상 (Intravital Imaging) **[데이터 해석]**
> > 살아있는 동물에서 면역세포의 움직임을 실시간으로 봅니다.
> >
> > **1. 혈관 투과성 실험 (P.19):**
> > * **설계:** 귀 진피(Ear dermis)에 **VEGFA164**(혈관내피성장인자) 주입. 혈관 내에는 **2000 kDa FITC-Dextran**(녹색) 주입.
> > * **결과:** VEGFA 주입 후 혈관 직경이 커지고, 녹색 형광이 혈관 밖으로 새어 나가는 **Vascular leakage**가 관찰됨.
> >
> > **2. 호중구의 박테리아 제거 (P.20):**
> > * **상황:** UPEC(대장균) 감염 모델.
> > * **관찰:** 0분부터 **150분**까지 촬영. 호중구(Neutrophil)가 몰려와 박테리아(형광)를 포식하여 제거(Clearance)하는 과정을 시각화.
> >
> > **3. 림프절 T세포 이동 (P.21):**
> > * **표지:** T세포는 **CFSE (Green)**로, 고내피세동맥(HEV)은 **Red**로 염색.
> > * **결과:** T세포가 HEV를 통해 림프절로 들어와 이동하는 3차원 궤적(Trafficking)을 확인.


> [!NOTE] Pages 22-27: 유전체 시각화 (FISH) 및 염색질 구조
> 유전자의 위치를 눈으로 확인하고, 유전자가 감겨 있는 히스톤의 상태를 분석하는 파트입니다.
>
> > [!INFO] PAGE 22-23: 3-D FISH (Fluorescence in situ hybridization)
> > * **정의:** 형광 RNA 탐침(Probe)을 DNA에 붙여서 염색체의 3차원적 구조와 거리를 측정합니다.
> > * **거리 측정 데이터 (P.23):**
> >     * **T cells:** $V_H$ (Red)와 $V_{H}J558$ (Green) 유전자 거리가 멉니다(Ready discriminated). 아직 재조합 전이라 그렇습니다.
> >     * **Pro-B cells:** 면역글로불린 유전자좌(Ig locus)가 **수축(Contraction)**합니다. 두 유전자(Red/Green)가 가까워져서 색이 겹쳐 보일 정도입니다. 이게 유전자 재조합의 전조 현상입니다.
>
> > [!IMPORTANT] PAGE 24: 만성 골수성 백혈병 (CML)과 필라델피아 염색체 **[병리 연계]**
> > 이 그림은 족보 중의 족보입니다.
> > * **정상:** 9번(Green)과 22번(Red) 염색체는 떨어져 있어야 합니다.
> > * **CML (Chronic Myeloid Leukemia):**
> >     * 9번과 22번 사이에 **상호 전좌(Reciprocal Translocation)**가 일어납니다.
> >     * **결과:** 9번의 **ABL** 유전자와 22번의 **BCR** 유전자가 합쳐져 **BCR-ABL** 융합 유전자가 생깁니다.
> >     * **병리:** 이 융합 단백질은 **Tyrosine Kinase 활성**이 비정상적으로 높아져 암(Tumor)을 유발합니다. (이걸 표적으로 한 약이 글리벡입니다.)
> >     * *그림 B/C를 보면 Red/Green 점이 붙어서 노란색/중간색으로 보이는 게 전좌의 증거입니다.*
>
> > [!INFO] PAGE 26-27: 염색질 구조 (Chromatin Structure)
> > * **구조:** DNA는 히스톤(Histone)을 감아 **뉴클레오솜(Nucleosome)**을 형성합니다. (Beads on a string)
> > * **히스톤 꼬리 변형 (Histone Tail Modification):**
> >     * **Acetylation (아세틸화):** 히스톤과 DNA 결합을 느슨하게 함 -> **Open Chromatin** -> **Gene Active**. (전사 기구가 접근 가능)
> >     * **Methylation (메틸화):** 주로 **Closed Chromatin** -> **Gene Inactive**. (단, 위치에 따라 다름)

> [!NOTE] Pages 28-32: 단백질-DNA 상호작용 분석 (ChIP & 3C)
> "이 단백질이 DNA 어디에 붙었니?"(ChIP)와 "이 DNA가 저 DNA랑 만났니?"(3C)를 묻는 기술입니다.
>
> > [!IMPORTANT] PAGE 28-30: ChIP (Chromatin Immunoprecipitation) **[실험 과정 암기]**
> > 단백질-DNA 결합을 분석하는 표준 기법입니다.
> >
> > **1. Workflow (P.29):**
> >     * **Cross-linking:** 포름알데히드로 단백질과 DNA를 딱 붙임(Fixation).
> >     * **Fragmentation:** DNA를 잘게 조각냄.
> >     * **Immunoprecipitation (IP):** 특정 단백질에 대한 **항체(Antibody)**로 낚아챔.
> >     * **DNA Extraction & Analysis:** 단백질을 녹여 없애고 DNA만 남겨서 PCR이나 Sequencing으로 분석.
> >     * *결과값: Enrichment = X/R (Input 대비 얼마나 농축됐는가)*
> >
> > **2. 주요 항체 리스트 (P.30 표):** *이거 구분 못하면 해석 못 합니다.*
> >     * **Active Markers (전사 활성):** H3K4me3 (Trimethyl), H3K9ac, H3K14ac (Acetylation은 주로 활성).
> >     * **Repressive Markers (전사 억제):** **H3K9me3**, **H3K27me3**. (꽉 뭉쳐서 유전자 꺼짐)
>
> > [!INFO] PAGE 31-32: 3C (Chromosome Conformation Capture)
> > 멀리 떨어진 DNA끼리의 상호작용(Looping 등)을 봅니다.
> > * **핵심 과정:** Cross-link -> **Restriction Enzyme (제한효소) 절단** -> **Ligation (연결)**.
> > * *포인트: 원래 멀리 있던 DNA라도, 단백질에 의해 묶여서 가까이 있다면 Ligation 되어 하나의 가닥으로 연결됩니다.*
> > * 결과물: Ligation products를 PCR/Sequencing하여 "아, 1번 염색체의 Promoter가 멀리 있는 Enhancer랑 붙어먹었구나"를 알아냅니다.

> [!NOTE] Pages 33-41: 유전자 가위 (CRISPR-Cas9)
> 2020년 노벨 화학상의 주인공입니다. 단순한 가위질을 넘어 유전자 조절까지 확장된 최신 경향을 파악하세요.
>
> > [!INFO] PAGE 33-35: 역사 (Timeline)
> > * **1987:** 대장균(E. coli)에서 CRISPR 서열(Clustered repeats) 최초 발견 (이땐 뭔지 몰랐음).
> > * **2012:** **Doudna & Charpentier** (노벨상 주역)가 Cas9의 DNA 절단 기전 규명.
> > * **2013:** 인간 세포(Mammalian cells)에서 게놈 교정 성공.
> > * **2020:** 노벨 화학상 수상.
>
> > [!IMPORTANT] PAGE 36-37: 작동 기전 (Mechanism) **[핵심 원리]**
> > 박테리아의 적응 면역 시스템에서 유래했습니다.
> >
> > **1. 구성 요소:**
> >     * **Cas9 Nuclease:** 가위(Enzyme).
> >     * **sgRNA (single guide RNA):** **crRNA** (Target 인식 20nt) + **tracrRNA** (Cas9 결합 구조)를 하나로 합친 것.
> >     * **PAM (Protospacer Adjacent Motif):** **5'-NGG-3'**. Cas9이 DNA를 자르려면 타겟 서열 바로 뒤에 무조건 이 NGG가 있어야 합니다. (없으면 못 자름!)
> >
> > **2. 수선 기전 (Repair Pathways):** 잘린 DNA(DSB)는 두 가지로 수선됩니다.
> >     * **NHEJ (Non-homologous end joining):** 대충 갖다 붙임 -> 돌연변이(Indel) 발생 -> **Gene Knockout (유전자 파괴)**.
> >     * **HDR (Homology-directed repair):** 정상 DNA 조각(Donor DNA)을 넣어줌 -> 그걸 보고 베껴서 수선 -> **Gene Correction (유전자 교정/삽입)**.
>
> > [!IMPORTANT] PAGE 38-39: Cas9 구조와 변형 (dCas9) **[상세 구조]**
> > **1. Cas9의 도메인:**
> >     * **RuvC Domain:** 비표적 가닥(Non-target strand) 절단.
> >     * **HNH Domain:** 표적 가닥(Target strand) 절단.
> >     * 이 두 도메인이 모두 작동해야 이중 나선이 잘립니다(DSB).
> >
> > **2. dCas9 (dead Cas9): 가위 기능을 죽인 버전**
> >     * **변이(Mutation):** **D10A** (RuvC 불활성) + **H840A** (HNH 불활성).
> >     * **기능:** DNA를 자르지는 못하고, 가이드 RNA를 따라 **붙기만 합니다(Binding only)**.
> >     * **Nickase:** 둘 중 하나만 죽이면(예: D10A만), 한 가닥만 자르는 Nickase가 됩니다.
>
> > [!TIP] PAGE 40: dCas9의 응용 (CRISPRi & Epigenetics)
> > 자르지 못하는 dCas9에 다른 효소(Effector)를 달아서 씁니다.
> > * **CRISPRi (Interference):** dCas9이 프로모터에 딱 붙어서 전사 기구가 못 오게 막음 -> **Transcription OFF**.
> > * **Epigenetic Editing:**
> >     * **dCas9 + EZH2:** EZH2는 **H3K27 Methyltransferase**입니다. 타겟 유전자의 히스톤을 메틸화시켜 영구적으로 꺼버립니다(Silencing).
> >     * **dCas9 + GFP:** 특정 유전자 위치를 형광으로 추적(Live imaging).

> [!NOTE] Pages 42-51: 동물 실험 기초 및 형질전환 동물
> "왜 쥐를 쓰는가?"부터 "어떻게 유전자를 넣는가?"까지, 동물 실험의 A to Z입니다.
>
> > [!INFO] PAGE 42-43: 동물 실험 윤리 (3Rs)
> > 연구자의 기본 소양입니다. 무조건 외우세요.
> > * **Reduce:** 사용되는 동물의 **수(Number)**를 줄입니다.
> > * **Refine:** 동물에게 가해지는 **고통/스트레스(Stress)**를 최소화하도록 실험법을 개선합니다.
> > * **Replace:** 동물 실험을 세포 실험이나 시뮬레이션 등 **다른 방법(Other methods)**으로 대체합니다.
>
> > [!IMPORTANT] PAGE 44-45: 근교계 (Inbred Strains) **[필수 암기]**
> > 실험 편차를 줄이기 위해 유전자가 99% 이상 동일한 "순계" 마우스를 씁니다.
> > * **생성법:** **형매교배(Brother-sister mating)**를 **20세대** 이상 반복합니다.
> > * **주요 Strain 특성 (P.45 표):** *이거 족보입니다.*
> >     * **BALB/c:** 방사선 감수성(Sensitivity), 하이브리도마(Hybridoma) 제조에 사용.
> >     * **C57BL/6:** 방사선 조사 후 간암(Hepatoma) 발생 빈도 높음, 높은 보체(Complement) 활성.
> >     * **C3H/HeJ:** 유선 종양(Mammary tumor) 발생 빈도 높음.
> >     * **AKR/J:** 백혈병(Leukemia) 발생 빈도 높음.
> >     * **CBA/N:** X-linked 면역결핍(Immunodeficiency).
>
> > [!INFO] PAGE 46-47: Congenic Strains & Adoptive Transfer
> > * **Congenic (동계):** 딱 **하나의 유전자좌(Locus)**만 다르고 나머지 모든 유전자가 동일한 쥐입니다. 특정 유전자의 기능만 콕 집어 연구할 때 씁니다.
> > * **Adoptive Transfer (양자 전달):** 같은 Inbred strain끼리는 거부 반응이 없죠? 그래서 한 쥐의 림프구(Lymphocyte)를 분리해서 다른 쥐에게 주사하여 면역 반응을 전달하는 실험입니다.
>
> > [!IMPORTANT] PAGE 48-51: Transgenic Animals (형질전환) 생성법
> > 외부 유전자(Transgene)를 인위적으로 삽입하는 기술입니다.
> > * **핵심 과정 (Microinjection):**
> >     1.  수정란(Fertilized egg)의 **전핵(Pronucleus)**에 DNA를 직접 찔러 넣습니다.
> >     2.  이 수정란을 **가임신(Pseudopregnant)** 대리모의 난관에 이식합니다.
> >     3.  태어난 새끼의 **10~30%** 정도만 유전자를 가집니다.
> > * **한계점 (P.51):** 유전자가 게놈의 **무작위 위치(Randomly)**에 끼어들어갑니다. 엉뚱한 곳에 들어가서 암을 유발하거나(Tumor), 발현이 안 될 수도 있습니다.

> [!NOTE] Pages 52-55: 유전자 적중 (Gene Targeting / Knockout)
> Transgenic의 한계(무작위 삽입)를 극복하고, 원하는 유전자만 정확히 없애거나 교체하는 기술입니다. **"Homologous Recombination (상동 재조합)"**이 핵심입니다.
>
> > [!IMPORTANT] PAGE 52-54: 선별 과정 (Positive/Negative Selection) **[기전 상세]**
> > ES cell(배아줄기세포)에 유전자를 넣고, 제대로 들어간 놈만 어떻게 골라낼까요? **Neo**와 **TK** 유전자를 씁니다.
> >
> > 1.  **구성:** 타겟 유전자 중간에 **NeoR**(항생제 내성)을 넣고, 끝부분에 **tkHSV**(자살 유전자)를 달아둡니다.
> > 2.  **경우의 수 (P.54):**
> >     * **Case A (Homologous Recombination - 성공):** 타겟 위치에 정확히 교체됨. **NeoR은 들어가고, TK는 떨어져 나감.**
> >     * **Case B (Random Insertion - 실패):** 엉뚱한 곳에 통째로 들어감. **NeoR도 있고, TK도 있음.**
> >     * **Case C (No Insertion):** 아무것도 안 들어감.
> > 3.  **약물 처리 결과:**
> >     * **Neomycin 처리:** NeoR이 없는 Case C 사망.
> > 	* **Ganciclovir 처리:** TK가 있는 Case B 사망 (Ganciclovir가 TK에 의해 독성물질로 변함).
> >     * **최종 생존:** **Case A (성공한 세포)**만 살아남습니다.
>
> > [!INFO] PAGE 55: 키메라(Chimera) 마우스
> > * 유전자가 조작된 ES cell(검은 쥐 유래)을 흰 쥐의 배반포(Blastocyst)에 넣습니다.
> > * 태어난 쥐는 흰색/검은색이 섞인 **키메라**가 됩니다. 이 쥐를 다시 교배해서 온몸이 검은색인(생식세포까지 전달된) 쥐를 선별해야 완전한 Knockout 마우스가 됩니다.

> [!NOTE] Pages 56-63: Cre-Lox 시스템 (Inducible Deletion)
> 유전자를 그냥 없애버리면 태어나자마자 죽을 수도 있습니다(Lethal). 그래서 **"특정 시기, 특정 조직에서만"** 유전자를 없애는 고급 기술이 필요합니다.
>
> > [!IMPORTANT] PAGE 56-59: Cre-LoxP 원리 **[핵심 원리]**
> > * **도구:**
> >     * **Cre Recombinase:** P1 파지에서 유래한 효소. **loxP** 자리를 인식해 그 사이를 잘라냅니다.
> >     * **loxP:** 34bp의 특정 DNA 서열. 타겟 유전자 양옆에 붙여둡니다(Flanked).
> > * **전략 (Tissue-specific Knockout):**
> >     * 쥐 A: **DNA Polymerase $\beta$** 유전자 양옆에 **loxP**를 심어둠 (Floxed mouse).
> >     * 쥐 B: **Thymus-specific promoter** 뒤에 **Cre** 유전자를 달아둠.
> >     * 교배(A x B): 태어난 쥐는 온몸에 loxP가 있지만, 오직 **흉선(Thymus)**에서만 Cre가 나와서 해당 유전자를 잘라버립니다.
> >     * *결과: 다른 조직은 멀쩡하고, 흉선에서만 유전자가 결손됩니다.*
>
> > [!TIP] PAGE 60: Cre-Lox Turn-on (발현 유도)
> > 반대로 유전자를 켤 수도 있습니다.
> > * **전략:** **Promoter - [loxP - STOP - loxP] - Gene**
> > * 평소에는 STOP 코돈 때문에 발현이 안 됩니다.
> > * Cre가 작동하면 **STOP 코돈이 잘려 나가면서(Deletion)** 유전자가 켜집니다. 독성 유전자(Toxic gene) 연구에 씁니다.
>
> > [!IMPORTANT] PAGE 61-62: Cre-ER (Tamoxifen 유도) **[약물 기전]**
> > 조직 특이성에 **시간(Timing)** 조절까지 더한 기술입니다.
> > * **구조:** Cre 효소에 **Estrogen Receptor (ER)**를 붙여놨습니다.
> > * **평소 (Without Tamoxifen):** HSP90이라는 단백질이 Cre-ER를 잡고 있어서 핵 안으로 못 들어갑니다(Inactive).
> > * **약물 처리 (With Tamoxifen):** **Tamoxifen**을 주사하면 ER에 결합하고, HSP90이 떨어져 나갑니다.
> > * **결과:** 자유로워진 Cre-ER이 **핵(Nucleus)**으로 이동해 loxP를 자릅니다.
> > * *이로써 연구자가 원할 때(Time-specific) 유전자를 조작할 수 있게 됩니다.*
